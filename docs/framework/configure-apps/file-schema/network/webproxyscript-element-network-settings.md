# <a name="ltwebproxyscriptgt-element-network-settings"></a>&lt;webProxyScript&gt;要素 (ネットワーク設定)
Web プロキシを検出するためのスクリプトの特性を構成します。  
  
 \<configuration>  
\<system.net >  
\<設定 >  
\<webProxyScript >  
  
## <a name="syntax"></a>構文  
  
```xml  
<webProxyScript  
  downloadTimeout="hh:mm:ss"  
/>  
```  
  
## <a name="attributes-and-elements"></a>属性および要素  
 以降のセクションでは、属性、子要素、および親要素について説明します。  
  
### <a name="attributes"></a>属性  
  
|属性|説明|  
|---------------|-----------------|  
|`downloadTimeout`|時間、分、および秒でスクリプトをダウンロードする最大時間を指定します。 既定値は、1 分です。|  
  
### <a name="child-elements"></a>子要素  
 なし。  
  
### <a name="parent-elements"></a>親要素  
  
|要素|説明|  
|-------------|-----------------|  
|[設定](../../../../../docs/framework/configure-apps/file-schema/network/settings-element-network-settings.md)|<xref:System.Net> 名前空間の基本的なネットワーク オプションを構成します。|  
  
## <a name="remarks"></a>コメント  
  
## <a name="configuration-files"></a>構成ファイル  
 この要素は、アプリケーション構成ファイルまたはマシン構成ファイル (Machine.config) で使用できます。
