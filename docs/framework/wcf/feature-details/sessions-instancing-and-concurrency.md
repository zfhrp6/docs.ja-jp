## <a name="concurrency"></a><span data-ttu-id="edcdd-149">同時実行</span><span class="sxs-lookup"><span data-stu-id="edcdd-149">Concurrency</span></span>  
 <span data-ttu-id="edcdd-150">同時実行は、 <xref:System.ServiceModel.InstanceContext> 内で同時にアクティブになるスレッドの数を制御します。</span><span class="sxs-lookup"><span data-stu-id="edcdd-150">Concurrency is the control of the number of threads active in an <xref:System.ServiceModel.InstanceContext> at any one time.</span></span> <span data-ttu-id="edcdd-151">同時実行を制御するには、<xref:System.ServiceModel.ServiceBehaviorAttribute.ConcurrencyMode%2A?displayProperty=nameWithType> と <xref:System.ServiceModel.ConcurrencyMode> 列挙値を使用します。</span><span class="sxs-lookup"><span data-stu-id="edcdd-151">This is controlled by using the <xref:System.ServiceModel.ServiceBehaviorAttribute.ConcurrencyMode%2A?displayProperty=nameWithType> with the <xref:System.ServiceModel.ConcurrencyMode> enumeration.</span></span>  
  
 <span data-ttu-id="edcdd-152">選択可能な同時実行モードは次の 3 つです。</span><span class="sxs-lookup"><span data-stu-id="edcdd-152">The following three concurrency modes are available:</span></span>
