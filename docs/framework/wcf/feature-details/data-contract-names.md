## <a name="basic-rules"></a><span data-ttu-id="6d3fb-107">基本的な規則</span><span class="sxs-lookup"><span data-stu-id="6d3fb-107">Basic Rules</span></span>

-   <span data-ttu-id="6d3fb-109">データ コントラクトの完全修飾名は、名前空間と名前から構成されます。</span><span class="sxs-lookup"><span data-stu-id="6d3fb-109">A fully-qualified data contract name consists of a namespace and a name.</span></span>
