|||  
|-|-|  
|ID|4812|  
|キーワード|探索|  
|レベル|警告|

## <a name="description"></a>説明  
 このイベントは、ReplyTo アドレスがなかったことが原因で探索要求メッセージが DiscoveryClient によってドロップされたときに生成されます。  
  
## <a name="message"></a>メッセージ  
 messageId='%1' の探索要求メッセージは、ReplyTo アドレスがなかったため、ドロップされました。
