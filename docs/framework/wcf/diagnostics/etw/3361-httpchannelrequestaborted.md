# 3361 - HttpChannelRequestAborted
## プロパティ  
  
|||  
|-|-|  
|ID|3361|  
|キーワード|HTTP|  
|レベル|警告|  
|チャネル|Microsoft\-Windows\-Application Server\-Applications\/Analytic|  
  
## 説明  
 このイベントは、HTTP チャネルの要求が中止されたときに生成されます。  
  
## メッセージ  
 HTTP チャネルの要求が中止されました。
