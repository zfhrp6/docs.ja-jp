# System.ServiceModel.TxCompletionStatusRemainsAttached
指定された操作の指定されたトランザクションは、InstanceContext に添付されたままになっています。  
  
## 説明  
 ServiceOperation が完了したのにトランザクションが完了しない場合にトレースされます。
