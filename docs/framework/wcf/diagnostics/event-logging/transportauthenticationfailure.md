# TransportAuthenticationFailure
ID : 174  
  
 重大度 : エラー  
  
 カテゴリ : SecurityAudit  
  
## 説明  
 このイベントはトランスポート認証が失敗したことを示し、クライアント ID、およびアクティビティ ID を表示します。
