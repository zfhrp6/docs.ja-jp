# FailFast
ID : 102  
  
 重大度 : エラー  
  
 カテゴリ : トレース  
  
## 説明  
 FailFast が呼び出されました。イベントには、メッセージ、スタック トレース、プロセス名、およびプロセス ID が表示されます。
