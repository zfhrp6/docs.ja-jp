# <a name="failedtoremoveperformancecounter"></a>FailedToRemovePerformanceCounter
ID : 119  
  
 重大度 : エラー  
  
 カテゴリ : ServiceModel  
  
## <a name="description"></a>説明  
 このイベントは、パフォーマンス カウンターが削除されなかったことを示します。 イベントには、カテゴリ名、カウンター名、インスタンス名、例外、プロセス名、およびプロセス ID が表示されます。
