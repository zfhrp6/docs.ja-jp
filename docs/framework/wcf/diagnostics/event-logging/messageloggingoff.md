# <a name="messageloggingoff"></a>MessageLoggingOff
Id: 117  
  
 重大度 : エラー  
  
 カテゴリ : ServiceModel

 このイベントは、メッセージ ログが無効になっていることを示します。 イベントには、プロセス名とプロセス ID が表示されます。
