### <a name="the-xaml-node-stream-concept"></a><span data-ttu-id="cde05-121">XAML ノード ストリームの概念</span><span class="sxs-lookup"><span data-stu-id="cde05-121">The XAML Node Stream Concept</span></span>

 <span data-ttu-id="cde05-126">XAML ノード ストリームの概念には、ノード ストリーム全体を通過すると、XAML 表現全体が処理されたことが保証されるという大きな利点があります。情報を処理するためのクエリ、DOM の操作、またはその他の非線形アプローチで、完全な XAML 表現の一部が不足していると心配する必要はありません。</span><span class="sxs-lookup"><span data-stu-id="cde05-126">The XAML node stream concept has the notable advantage that if you go through the entire node stream, you are assured that you have processed the entire XAML representation;
