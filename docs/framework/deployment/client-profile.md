<span data-ttu-id="b5e05-103">.NET Client Profile は、.NET Framework のサブセットで、[!INCLUDE[net_v40_short](../../../includes/net-v40-short-md.md)] 以前のバージョンに含まれ、クライアント アプリケーション用に最適化されていました。</span><span class="sxs-lookup"><span data-stu-id="b5e05-103">The .NET Client Profile is a subset of the .NET Framework, which was provided with [!INCLUDE[net_v40_short](../../../includes/net-v40-short-md.md)] and earlier versions and was optimized for client applications.</span></span> <span data-ttu-id="b5e05-104">.NET Framework は、Windows、Windows Phone、および Microsoft Azure 用の開発プラットフォームであり、マネージ アプリの実行環境と .NET Framework クラス ライブラリを提供します。</span><span class="sxs-lookup"><span data-stu-id="b5e05-104">The .NET Framework is a development platform for Windows, Windows Phone and Microsoft Azure and provides a managed app execution environment and the .NET Framework class library.</span></span> <span data-ttu-id="b5e05-105">[!INCLUDE[net_v40_short](../../../includes/net-v40-short-md.md)] 以前のバージョンでは、2 種類の配置オプション (完全な .NET Framework と Client Profile) が用意されていました。</span><span class="sxs-lookup"><span data-stu-id="b5e05-105">The [!INCLUDE[net_v40_short](../../../includes/net-v40-short-md.md)] and earlier versions provided two deployment options: the full .NET Framework and the Client Profile.</span></span> <span data-ttu-id="b5e05-106">完全な .NET Framework に比べて、Client Profile では、配置の迅速化とアプリ インストール パッケージの小型化が実現されました。</span><span class="sxs-lookup"><span data-stu-id="b5e05-106">The Client Profile enabled faster deployment and smaller app installation packages than the full .NET Framework.</span></span>  
  
 <span data-ttu-id="b5e05-107">[!INCLUDE[net_v45](../../../includes/net-v45-md.md)] 以降では、Client Profile が中止され、完全な再頒布可能パッケージのみが使用できます。</span><span class="sxs-lookup"><span data-stu-id="b5e05-107">Starting with the [!INCLUDE[net_v45](../../../includes/net-v45-md.md)], the Client Profile has been discontinued and only the full redistributable package is available.</span></span> <span data-ttu-id="b5e05-108">より小さいダウンロード サイズや高速化された配置などの [!INCLUDE[net_v45](../../../includes/net-v45-md.md)] の最適化により、別の配置パッケージが不要になりました。</span><span class="sxs-lookup"><span data-stu-id="b5e05-108">Optimizations provided by the [!INCLUDE[net_v45](../../../includes/net-v45-md.md)], such as smaller download size and faster deployment, have eliminated the need for a separate deployment package.</span></span> <span data-ttu-id="b5e05-109">単一の再頒布可能パッケージにより、インストール プロセスが効率化され、アプリケーションの配置オプションが簡略化されます。</span><span class="sxs-lookup"><span data-stu-id="b5e05-109">The single redistributable streamlines the installation process and simplifies your app's deployment options.</span></span>  
  
 <span data-ttu-id="b5e05-110">ただし、.NET Framework 4 または 3.5 を対象とする場合、Client Profile の概要と用途については、.NET Framework 4 ドキュメントの「[.NET Framework Client Profile](http://msdn.microsoft.com/library/cc656912\(v=vs.100\).aspx)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="b5e05-110">However, if you are targeting the .NET Framework 4 or 3.5 and want to learn more about the Client Profile and when to use it, see [.NET Framework Client Profile](http://msdn.microsoft.com/library/cc656912\(v=vs.100\).aspx) in the .NET Framework 4 documentation.</span></span>
