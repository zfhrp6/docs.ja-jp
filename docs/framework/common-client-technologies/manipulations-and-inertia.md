# <a name="manipulations-and-inertia"></a>操作と慣性
"*操作*" および "*慣性*" プロセッサ クラスは、アプリケーションが自然で直感的な方法で画面上のユーザー インターフェイス (UI) 要素を移動するのを支援します。 操作では、ユーザーはマウス デバイス、スタイラス、(直接タッチ対応シナリオでの) 指などのマニピュレーターを使って、コンポーネントの移動、回転、サイズ変更を行うことができます。 "*慣性*" は、要素上で摩擦力をシミュレートして、動いている UI 要素の実際の動作をエミュレートします。 これにより、要素は (直線運動と角運動の両方で) 徐々に動作の速度を遅らせてから停止します。  
  
 操作と慣性の処理は、Microsoft Windows Presentation Foundation (WPF) や Microsoft XNA などのさまざまな UI フレームワークで使うことができます。
