# <a name="imetadatatablesgetcolumn-method"></a>IMetaDataTables::GetColumn メソッド
指定した列と、指定されたテーブルの行のセルに含まれる値へのポインターを取得します。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT GetColumn (   
    [in]  ULONG   ixTbl,  
    [in]  ULONG   ixCol,  
    [in]  ULONG   rid,  
    [out] ULONG   *pVal  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `ixTbl`  
 [in]テーブルのインデックス。  
  
 `ixCol`  
 [in]テーブル内の列のインデックス。  
  
 `rid`  
 [in]テーブル内の行のインデックス。  
  
 `pVal`  
 [out]セルの値へのポインター。
