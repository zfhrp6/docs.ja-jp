# <a name="iceegengenerateceefile-method"></a>ICeeGen::GenerateCeeFile メソッド
これに現在読み込まれているコード ベースを含むコード ベースのファイルを生成[ICeeGen](../../../../docs/framework/unmanaged-api/metadata/iceegen-interface.md)インターフェイスです。  
  
 このメソッドは、古いは使用できません。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT GenerateCeeFile ();  
```  
  
#### <a name="parameters"></a>パラメーター  
 `GenerateCeeFile` パラメーターはありません。
