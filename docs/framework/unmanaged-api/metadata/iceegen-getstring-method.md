# <a name="iceegengetstring-method"></a>ICeeGen::GetString メソッド
指定の相対仮想アドレスに格納された文字列を取得します。  
  
 このメソッドは、古いは使用できません。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT GetString (  
    [in]  ULONG      RVA,   
    [out] LPWSTR     *lpString  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `RVA`  
 [in]返される文字列の相対仮想アドレス。  
  
 `lpString`  
 [out]返される文字列。
