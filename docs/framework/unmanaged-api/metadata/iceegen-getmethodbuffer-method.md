# <a name="iceegengetmethodbuffer-method"></a>ICeeGen::GetMethodBuffer メソッド
指定の相対仮想アドレスのメソッドを適切なサイズのバッファーを取得します。  
  
 このメソッドは、古いは使用できません。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT GetMethodBuffer (  
    [in]  ULONG       RVA,  
    [out] UCHAR       **lpBuffer  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `RVA`  
 [in]バッファーを取得するメソッドの相対仮想アドレス。  
  
 `lpBuffer`  
 [out]返されたバッファーへのポインター。
