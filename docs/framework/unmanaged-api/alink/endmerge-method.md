# <a name="endmerge-method"></a>EndMerge メソッド
すべてのカスタム属性が生成スコープにマージされていることを示します。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT EndMerge(  
    mdAssembly AssemblyID  
) PURE;  
```  
  
#### <a name="parameters"></a>パラメーター  
 `AssemblyID`  
 アセンブリの ID です。  
  
## <a name="return-value"></a>戻り値  
 メソッドが成功した場合は、S_OK を返します。  
  
## <a name="requirements"></a>要件  
 Alink.h が必要です。
