# <a name="icordebugheapvalue2-interface1"></a>ICorDebugHeapValue2 Interface1
共通言語ランタイム (CLR) 処理のサポートを提供する ICorDebugHeapValue の拡張子です。  
  
## <a name="methods"></a>メソッド  
  
|メソッド|説明|  
|------------|-----------------|  
|[CreateHandle メソッド](../../../../docs/framework/unmanaged-api/debugging/icordebugheapvalue2-createhandle-method.md)|このため、指定した型のハンドルを作成`ICorDebugHeapValue2`オブジェクト。|  
  
## <a name="remarks"></a>コメント  
  
> [!NOTE]
>  このインターフェイスは、コンピューター間またはプロセス間でのリモート呼び出しをサポートしていません。
