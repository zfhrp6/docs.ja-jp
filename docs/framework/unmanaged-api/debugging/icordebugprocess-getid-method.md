# <a name="icordebugprocessgetid-method"></a>ICorDebugProcess::GetID メソッド
プロセスのオペレーティング システム (OS) の ID を取得します。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT GetID([out] DWORD *pdwProcessId);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `pdwProcessId`  
 [out]プロセスの一意の ID。
