# <a name="inotifysource2-interface"></a>INotifySource2 インターフェイス

|メソッド|説明|  
|------------|-----------------|  
|[SetNotifyFilter メソッド](../../../../docs/framework/unmanaged-api/diagnostics/inotifysource2-setnotifyfilter-method.md)|このソースで使用するための通知フィルターが割り当てられます。|
