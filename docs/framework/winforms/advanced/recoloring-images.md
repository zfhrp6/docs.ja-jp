# <a name="recoloring-images"></a>イメージの色の変更
色の変更は、イメージの色を調整するプロセスです。 色の変更の例をいくつかは、別、別の色の基準とした色の輝度を調整する、明るさやすべての色コントラストを調整および色をグレースケールに変換する 1 つの色を変更します。
