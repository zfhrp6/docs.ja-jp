# <a name="byref-visual-basic"></a>ByRef (Visual Basic)
呼び出されたプロシージャが呼び出し元のコードで引数の基になる変数の値を変更できるように引数が渡されることを指定します。  
  
## <a name="remarks"></a>コメント  
 `ByRef` 修飾子は、次のコンテキストで使用できます。
