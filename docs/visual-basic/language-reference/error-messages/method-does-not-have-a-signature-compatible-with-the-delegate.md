# Method does not have a signature compatible with the delegate
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

メソッドのシグネチャと使用しようとしているデリゲートのシグネチャが対応していません。  `Delegate` ステートメントでは、デリゲート クラスのパラメーターの型と戻り値の型を定義します。  型が対応しているパラメーターと戻り値を使用したプロシージャによって、このデリゲート型のインスタンスを作成できます。  
  
 **エラー ID**: BC36563
