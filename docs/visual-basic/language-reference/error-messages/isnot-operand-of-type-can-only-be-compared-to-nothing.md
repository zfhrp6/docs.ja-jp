# &#39;IsNot&#39; operand of type &#39;typename&#39; can only be compared to &#39;Nothing&#39;, because &#39;typename&#39; is a nullable type
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

`IsNot` 演算子を使用して、Null 許容として宣言されている変数を `Nothing` 以外の式と比較しました。  
  
 **エラー ID:** BC32128  
  
### このエラーを解決するには  
  
1.  `IsNot` 演算子を使用して、Null 許容型を `Nothing` 以外の式と比較するには、Null 許容型の `GetType` メソッドを呼び出して、その結果を式と比較します。次に例を示します。  
  
    ```vb#  
    Dim number? As Integer = 5  
  
    If number IsNot Nothing Then  
      If number.GetType() IsNot Type.GetType("System.Int32") Then   
  
      End If  
    End If  
    ```
