# <a name="labels-that-are-numbers-must-be-followed-by-colons"></a>数字を指定するラベルの後には、コロンが必要です。
行番号は、ラベルの他の種類と同じ規則に従うし、コロンを含める必要があります。  
  
 **エラー ID:** BC30801  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   配置後にコードの行の先頭のコロン数例えば：  
  
    ```  
    400:    X += 1  
    ```
