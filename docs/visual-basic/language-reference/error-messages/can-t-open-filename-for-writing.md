# Can&#39;t open &#39;&lt;filename&gt;&#39; for writing
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

指定されたファイルを書き込むために開くことができません。おそらくファイルが既に開いているためと思われます。  
  
 **Error ID:** BC2012  
  
### このエラーを解決するには  
  
1.  ファイルを一度閉じてから開いてください。  
  
2.  ファイルのアクセス許可を確認してください。
