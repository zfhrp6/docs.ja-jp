# <a name="can39t-open-39ltfilenamegt39-for-writing"></a>できます&#39;t はオープン&#39; &lt;filename&gt; &#39;の書き込み
作成するため、おそらく既に開かれているために、指定したファイルを開くことができません。  
  
 **エラー ID:** BC2012  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  ファイルを閉じて、再度開きます。  
  
2.  ファイルのアクセス許可を確認してください。
