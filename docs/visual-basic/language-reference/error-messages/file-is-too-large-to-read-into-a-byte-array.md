# <a name="file-is-too-large-to-read-into-a-byte-array"></a>バイト配列に読み取るにはファイルが大きすぎます。
バイト配列に読み取るしようとしているファイルのサイズは 4 GB を超えています。 `My.Computer.FileSystem.ReadAllBytes`メソッドは、このサイズを超えるファイルを読み取ることができません。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   使用して、<xref:System.IO.StreamReader>ファイルを読み取れません。 詳細については、次を参照してください。[の基礎の .NET Framework ファイルの I/O とファイル システム (Visual Basic)](../../../visual-basic/developing-apps/programming/drives-directories-files/basics-of-net-framework-file-io-and-the-file-system.md)です。
