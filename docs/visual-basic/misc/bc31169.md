# <a name="character-39character39-0xlthexadecimalgt-is-not-allowed-at-the-beginning-of-an-xml-name"></a>文字&#39;文字&#39;(0 x&lt;16 進&gt;) は、XML 名の先頭には許可されていません
サポートされていない文字が、XML リテラルまたは XML 軸プロパティの名前の先頭に含まれています。  
  
 **エラー ID:** BC31169  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   XML リテラルまたは XML 軸プロパティの名前の先頭から、サポートされていない文字を削除します。
