`System.AttributeUsageAttribute` なしで宣言された属性を使用して、その使用法を定義しようとしました。  
  
 **エラー ID:** BC31505  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  カスタム属性はから派生したクラスである必要があります`System.Attribute`がある、`AttributeUsageAttribute`属性を適用します。
