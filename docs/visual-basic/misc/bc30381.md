# <a name="39finally39-can-only-appear-once-in-a-39try39-statement"></a>&#39;最後に&#39;回のみ使用できますの&#39;再試行&#39;ステートメント
`Finally` は `Try...Catch...Finally` ブロックを終了するために使用します。そのため、ブロックの最後で一度だけ記述できます。  
  
 **エラー ID:** BC30381  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  不要な `Finally`を見つけて削除します。
