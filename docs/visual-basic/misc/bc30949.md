# <a name="array-initializer-cannot-be-specified-for-a-non-constant-dimension-use-the-empty-initializer-3939"></a><span data-ttu-id="ef1d7-102">非定数の次元の配列初期化子を指定することはできません。空の初期化子を使用します。&#39;{}&#39;</span><span class="sxs-lookup"><span data-stu-id="ef1d7-102">Array initializer cannot be specified for a non constant dimension; use the empty initializer &#39;{}&#39;</span></span>
<span data-ttu-id="ef1d7-103">配列が、コンパイル時に知られていないディメンションを初期化します。</span><span class="sxs-lookup"><span data-stu-id="ef1d7-103">An array initializes a dimension that is not known at compile time.</span></span>  
  
 <span data-ttu-id="ef1d7-104">次のコードでは、このエラーが生成されます。</span><span class="sxs-lookup"><span data-stu-id="ef1d7-104">The following code generates this error.</span></span>  
  
```  
Dim j As Integer  
Dim intArray As Integer = New Integer(1, j) {{0, 100}, {1,101}}  
```  
  
 <span data-ttu-id="ef1d7-105">次のコードでは、エラーが回避されます。</span><span class="sxs-lookup"><span data-stu-id="ef1d7-105">The following code avoids the error.</span></span>  
  
```  
Dim intArray As Integer = New Integer(1, j) {}  
For i As Integer = 0 To j  
    intArray(0, i) = i  
    intArray(1, i) = 100 + i  
Next i  
```
