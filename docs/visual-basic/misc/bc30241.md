# 名前付き引数が必要です。
プロシージャ呼び出しで引数リストは、名前別に引数を渡してから、位置別に引数を渡します。 すべての位置指定引数は、プロシージャ呼び出しで名前によって渡される引数の前に指定する必要があります。  
  
 **エラー ID:** BC30241  
  
### このエラーを解決するには  
  
-   引数リストを書き換えて、名前によって渡される引数の前に位置によって渡される引数を配置します。
