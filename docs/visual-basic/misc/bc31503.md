# <a name="39lttypenamegt39-cannot-be-used-as-an-attribute-because-it-is-not-a-class"></a>&#39;&lt;typename&gt; &#39;クラスではないため、属性として使用することはできません
クラスではない型を属性として使用しようとしました。  
  
 **エラー ID:** BC31503  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  `System.Attribute`から派生するクラスとしてカスタム属性を定義します。
