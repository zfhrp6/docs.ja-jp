# &lt;error&gt;: &#39;&lt;structurename1&gt;&#39; に &#39;&lt;structurename2&gt;&#39; が含まれています
構造体のメンバーにその構造体自体を指定すると、構造体の定義が循環します。  
  
 **エラー ID:** BC30293  
  
### このエラーを解決するには  
  
-   構造体メンバーの名前を変更します。
