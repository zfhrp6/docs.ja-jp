# <a name="39select39-statements-are-not-valid-in-the-immediate-window"></a>&#39;選択&#39;ステートメントは、イミディ エイト ウィンドウでは有効ではありません。
`Select` ステートメントは、ソース コードでのみ使用できます。  
  
 **エラー ID:** BC30718  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   デバッグ コードから `Select` ステートメントを削除します。
