# &#39;End AddHandler&#39; の前には、対応する &#39;AddHandler&#39; 宣言を指定しなければなりません
`End AddHandler` ステートメントが発生しましたが、対応する `AddHandler` ステートメントがありません。`End AddHandler` の前に、対応する `AddHandler` ステートメントが必要です。  
  
 **エラー ID:** BC31124  
  
### このエラーを解決するには  
  
-   先行する `AddHandler` ステートメントが有効であり、スペルが正しいことを確認します。
