# <a name="members-in-a-structure-cannot-be-declared-39ltspecifiergt39"></a>構造体のメンバーを宣言することはできません&#39;&lt;指定子&gt;&#39;
`Structure` ステートメント内のメンバーに対して無効な指定子を使用しました  
  
 **エラー ID:** BC30435  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定子を削除します。
