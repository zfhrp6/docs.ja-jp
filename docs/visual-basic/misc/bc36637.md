# <a name="the-3939-character-cannot-be-used-here"></a>&#39;しますか?&#39;文字はここでは使用できません
'?' 文字を使用して、値型または構造が Null を許容することを指定できます。 これ以外の場合に使用することは制限されています。 たとえば、次のコードではこの例外が発生します。  
  
```  
' Not valid.  
' #Const found = True?  
```  
  
 **エラー ID:** BC36637  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   宣言から '?' 文字を削除します。
