# 属性に Public コンストラクターがないため、この属性を使用できません
使用されている属性のコンストラクターは `Private` で、使用することはできません。  
  
 **エラー ID:** BC30758  
  
### このエラーを解決するには  
  
1.  開発したカスタム属性でこのエラーが発生する場合は、`Sub``New` コンストラクターのアクセス修飾子を `Public` に変更します。
