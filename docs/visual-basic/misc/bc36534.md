# <a name="expression-cannot-be-converted-into-an-expression-tree"></a>式を式のツリーに変換することはできません
ほとんどの式は式のツリーに変換できますが、変換できないものもあります。 このエラーは、変換できない式を変換しようとしたことを示しています。  
  
 **エラー ID:** BC36534
