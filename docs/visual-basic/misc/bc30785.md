# <a name="parameter-specifier-is-duplicated"></a>パラメーター指定子が重複しています
パラメーター指定子が、パラメーターに複数回適用されています。  
  
 **エラー ID:** BC30785  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   重複している指定子を削除します。
