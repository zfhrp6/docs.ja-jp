# <a name="enum-39ltenumnamegt39-must-contain-at-least-one-member"></a>Enum &#39; &lt;enumname&gt; &#39;には、少なくとも 1 つのメンバーを含める必要があります
空の列挙型があります。 少なくとも 1 つのメンバーを `Enum` ステートメントと `End Enum` ステートメントの間で宣言する必要があります。  
  
 **エラー ID:** BC30280  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  列挙体にメンバーの宣言を追加します。
