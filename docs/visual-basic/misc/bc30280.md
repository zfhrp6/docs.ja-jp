# 列挙型 &#39;&lt;enumname&gt;&#39; は少なくとも 1 つのメンバーを含んでいる必要があります
空の列挙型があります。 少なくとも 1 つのメンバーを `Enum` ステートメントと `End Enum` ステートメントの間で宣言する必要があります。  
  
 **エラー ID:** BC30280  
  
### このエラーを解決するには  
  
1.  列挙体にメンバーの宣言を追加します。
