# モジュールのプロパティを &#39;&lt;specifier&gt;&#39; として宣言することはできません
モジュール内のプロパティに `MustOverride`、`NotOverridable`、`Overridable` などの指定子を使用しようとしました。  
  
 **エラー ID:** BC30503  
  
### このエラーを解決するには  
  
-   指定子を削除します。
