# <a name="39get39-statements-are-no-longer-supported"></a>ステートメントの 'get' がサポートされていません
`Get`ステートメントはサポートされていません。 ファイル I/O 機能は、 `Microsoft.VisualBasic` 名前空間で使用できます。  
  
 `Get`ファイル操作はサポートされていませんし、プロパティ プロシージャの構文でのみ使用できます。  
  
 **エラー ID:** BC30829  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  ファイルのメンバーを操作する`System.IO`、 `FileSystemObject`、および[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]ランタイム関数です。
