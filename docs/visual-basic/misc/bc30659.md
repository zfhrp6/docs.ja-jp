# <a name="property-or-field-39ltnamegt39-does-not-have-a-valid-attribute-type"></a>プロパティまたはフィールド&#39;&lt;名前&gt;&#39;は有効な属性型がありません
正しい属性型を受け入れない属性コンストラクターを使用している可能性があります。しかし、属性には他の適切なコンストラクターがある可能性があります。  
  
 **エラー ID:** BC30659  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  この属性の使用法を確認します。
