# 匿名型は、少なくとも 1 つのメンバーを含んでいる必要があります
匿名型の宣言で初期化子リストを空にすることはできません。  
  
```  
' Not valid. ' Dim anonInstance = New With {}  
```  
  
 **エラー ID:** BC36574  
  
### このエラーを解決するには  
  
-   次のコードに示すように、中かっこ内のメンバーを宣言します。  
  
    ```  
    Dim anonInstance = New With {.MemberName = "value"}  
    ```
