# 型 &#39;&lt;type1&gt;&#39; および &#39;&lt;type2&gt;&#39; に対して演算子 &#39;&lt;operatorname&gt;&#39; が定義されていません
2 項演算子を使用して、値を返す 2 つの関連コード要素に無効な操作を実行しようとしました。  
  
 **エラー ID:** BC30452  
  
### このエラーを解決するには  
  
-   2 つの要素を確認し、互換性があるかどうかを確認します。
