# <a name="39custom-event39-must-end-with-a-matching-39end-event39"></a>&#39;カスタム イベント&#39;、対応する終了する必要があります&#39;終了イベント&#39;
`Custom Event` 宣言は、 `End Event` ステートメントで終了する必要があります。  
  
 **エラー ID:** BC31114  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `Custom Event` 宣言が `End Event` ステートメントで終了していることを確認します。
