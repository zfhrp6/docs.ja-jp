# &#39;Set&#39; パラメーターを &#39;&lt;specifier&gt;&#39; に宣言することはできません
`Set` プロパティ プロシージャのパラメーターに無効な修飾子が指定されました。  
  
 **エラー ID:** BC31065  
  
### このエラーを解決するには  
  
-   `ByRef`、`ParamArray`、`Optional` などの無効な修飾子を削除します。
