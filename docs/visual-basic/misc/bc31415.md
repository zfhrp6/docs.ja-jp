# &#39;&lt;typename&gt;&#39; は予約された名前であるため、&#39;&lt;implementsclause&gt;&#39; で &#39;&lt;typename&gt;&#39; を実装できません。
`Implements` ステートメントに、予約された名前が指定されました。  
  
 **エラー ID:** BC31415  
  
### このエラーを解決するには  
  
-   実装できないインターフェイス メンバーの名前を変更します。
