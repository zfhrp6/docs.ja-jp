# 構造体の中の &#39;Declare&#39; ステートメントを &#39;&lt;specifier&gt;&#39; と宣言することはできません
`Declare` 宣言に、`Structure` 宣言内で無効な指定子があります。  
  
 **エラー ID:** BC30791  
  
### このエラーを解決するには  
  
-   指定子を削除します。
