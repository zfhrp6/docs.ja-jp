# <a name="39declare39-statements-in-a-structure-cannot-be-declared-39ltspecifiergt39"></a>&#39;宣言&#39;構造でステートメントを宣言することはできません&#39;&lt;指定子&gt;&#39;
`Declare` 宣言に、 `Structure` 宣言内で無効な指定子があります。  
  
 **エラー ID:** BC30791  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定子を削除します。
