# デリゲートの実装に必要な、型 &#39;&lt;typename&gt;&#39; の delegate コンストラクターが見つかりません。
この型を含むアセンブリが破損しているか、または無効である可能性があります。  
  
 **エラー ID:** BC31074  
  
### このエラーを解決するには  
  
1.  この型を含むアセンブリが有効であることを確認します。  
  
2.  プログラムをもう一度コンパイルし、エラーがまだ発生するかどうか確認します。  
  
3.  エラーが引き続き発生する場合は、作業内容を保存し、[!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)] を再起動します。
