# &#39;End Set&#39; は、行の最初のステートメントでなければなりません
`End Set` ステートメントがコロン \(:\) ステートメント区切り文字の後に続いています。`End Set` はそのソース コード行でただ 1 つのステートメントである必要があります。  
  
 **エラー ID:** BC32034  
  
### このエラーを解決するには  
  
-   複数のステートメントを別々の行に分けます。
