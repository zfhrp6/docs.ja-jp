# &#39;&lt;keyword&gt;&#39; は、クラス内でのみ有効です
`Me` や `MyClass` など、クラスに関連するキーワードがクラス定義外で使用されています。  
  
 **エラー ID:** BC32002  
  
### このエラーを解決するには  
  
-   キーワードを使用するコードがクラス インスタンスに関連する場合は、これをクラスの実装に移動します。  
  
-   キーワードを使用するコードがクラスに適用されない場合は、無効なキーワードを削除します。
