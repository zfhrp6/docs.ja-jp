# <a name="overload-resolution-failed-because-no-accessible-39ltmethodgt39-can-be-called-with-these-argumentslterrorgt"></a>オーバー ロードの解決が失敗しましたない&#39;&lt;メソッド&gt;&#39;これらの引数と共に呼び出せる:&lt;エラー。&gt;
オーバーロードされたメソッドを呼び出しましたが、コンパイラで、対応する引数リストを持つメソッドを検出できません。  
  
 **エラー ID:** BC30518  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定した引数リストを確認します。
