# <a name="39end-operator39-must-be-the-first-statement-on-a-line"></a>&#39;End Operator&#39;行の最初のステートメントをする必要があります
`End Operator` ステートメントがコロン (`:`) ステートメント区切り文字の後に続いています。 `End Operator` はそのソース コード行でただ 1 つのステートメントである必要があります。  
  
 **エラー ID:** BC33006  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   複数のステートメントを別々の行に分けます。
