# Web メソッドを、デバッグ ウィンドウで評価することはできません
XML Web サービス メソッドは、デバッグ ウィンドウ内でテストできません。  
  
 **エラー ID:** BC30732  
  
### このエラーを解決するには  
  
-   コード エディターから XML Web サービス メソッドをテストします。
