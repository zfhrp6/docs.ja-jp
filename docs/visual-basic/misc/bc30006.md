# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-definition-for-event-39lteventnamegt39"></a>モジュールに必要な参照&#39; &lt;modulename&gt; &#39;イベントの定義を含む&#39; &lt;eventname&gt;&#39;
モジュールに必要な参照 '<`modulename`>' の定義イベントを含む' <`eventname`>' です。 プロジェクトに参照を追加してください。  
  
 プロジェクト内で直接参照されないモジュールでイベントが定義されています。 Visual Basic コンパイラでは、イベントが 1 つ以上のモジュールで定義されている場合に備えて、あいまいさを避けるために参照が必要です。  
  
 **エラー ID:** BC30006  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   参照されていないモジュールの名前をプロジェクト参照に含めます。
