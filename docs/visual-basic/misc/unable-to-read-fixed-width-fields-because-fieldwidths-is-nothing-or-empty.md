# FieldWidth が Nothing または empty であるため、固定幅フィールドを読み取れません
`FieldWidths` プロパティが `Nothing` に設定されているか空であるため、`TextFieldParser` は固定幅フィールドを読み取ることができません。  
  
### このエラーを解決するには  
  
-   `FieldWidths` を正しい値に設定します。
