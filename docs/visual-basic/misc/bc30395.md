# &#39;&lt;specifier&gt;&#39; は、Structure 宣言では使用できません。
`Dim` または `Override` などの指定子を使用していますが、`Structure` 宣言では使用できません。  
  
 **エラー ID:** BC30395  
  
### このエラーを解決するには  
  
-   指定子を削除します。
