# 型 &#39;&lt;typename1&gt;&#39; から型 &#39;&lt;typename2&gt;&#39; へのキャストはできません
変数の型を変換しようとしていますが、その変換はできません。  
  
### このエラーを解決するには  
  
1.  変換に関係する型をチェックし、意味のある型に変換しようとしていることを確認してください。
