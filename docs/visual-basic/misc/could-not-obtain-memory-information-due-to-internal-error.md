# 内部エラーのため、メモリ情報を取得できませんでした
`My.Computer.Info` オブジェクトのメモリ情報のプロパティの 1 つに対する呼び出しが失敗しました。  
  
### このエラーを解決するには  
  
-   `My.Computer.Info` オブジェクトのメモリ情報プロパティへの呼び出しの周囲に `Try...Catch` を追加します。
