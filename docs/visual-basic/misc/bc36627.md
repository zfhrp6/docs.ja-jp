# ラムダ式は、デバッガー内では評価できません
デバッグ セッション中に、ラムダ式が **\[ウォッチ\]** ウィンドウまたは **\[イミディエイト\]** ウィンドウに入力されました。 ラムダ式は、デバッグ ウィンドウ内では使用できません。  
  
 **エラー ID:** BC36627  
  
### このエラーを解決するには  
  
-   コード エディターでラムダ式をテストします。
