# <a name="error-embedding-win32-manifest-ltmanifestgt"></a>Win32 マニフェストの埋め込みエラー:&lt;マニフェスト&gt;
`/win32manifest` コンパイラ オプションに指定されたファイルを埋め込むことができません。 マニフェスト ファイルに対するセキュリティ特権が不十分であることが原因の可能性があります。  
  
 **エラー ID:** BC31191  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  Visual Basic コンパイラのコマンドを実行している ID が Win32 マニフェスト ファイルにアクセスできることを確認します。
