# 参照されたオブジェクト &#39;&lt;objectname&gt;&#39; の値が &#39;Nothing&#39; です
参照先オブジェクトがまだ設定されていないか、または値 `Nothing` に既に設定されています。  
  
 **エラー ID:** BC30705  
  
### このエラーを解決するには  
  
-   オブジェクトの名前のスペルが正しいことと、オブジェクトが設定されていることを確認します。
