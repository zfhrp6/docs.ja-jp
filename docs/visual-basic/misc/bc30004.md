# 文字定数には 1 文字のみを指定しなければなりません
文字定数が正しくありません。複数の文字であるため、あるいは null 文字 \(すべてのビットがゼロ\) であるためです。  
  
 **エラー ID:** BC30004  
  
### このエラーを解決するには  
  
-   文字定数を 1 文字に変更するか、null 文字を正しい文字に変更します。
