# 型 &#39;&lt;typename&gt;&#39; は定義されていないか、またはそれを含むモジュールがデバッグ セッションで読み込まれていません
使用できない、または定義されていない型が参照されました。  
  
 **エラー ID:** BC30743  
  
### このエラーを解決するには  
  
1.  型の名前のスペルが正しいことを確認します。  
  
2.  現在のクラスまたはモジュール内で型を定義します。
