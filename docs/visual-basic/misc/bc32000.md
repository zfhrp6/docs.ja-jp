# ローカル変数 &#39;&lt;variablename&gt;&#39; は、宣言されるまで参照できません
プロシージャ内の変数が、宣言される前に使用されています。  
  
 **エラー ID:** BC32000  
  
### このエラーを解決するには  
  
-   宣言を、変数が最初に参照される前に移動します。
