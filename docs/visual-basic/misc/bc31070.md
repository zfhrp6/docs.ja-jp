# インターフェイス内のクラスを &#39;&lt;specifier&gt;&#39; と宣言することはできません
インターフェイス内で宣言されたクラスで、正しくない修飾子が指定されました。  
  
 **エラー ID:** BC31070  
  
### このエラーを解決するには  
  
-   `Public`、`Private`、`Shared`、`Friend`、`Protected`、`Protected Friend`、または `Overridable` などの修飾子を削除します。
