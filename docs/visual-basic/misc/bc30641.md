# <a name="39byval39-and-39byref39-cannot-be-combined"></a>&#39;です。ByVal &#39;および &#39;です。ByRef &#39;組み合わせることができません。
`ByVal` キーワードと `ByRef` キーワードは相互に排他的です。  
  
 **エラー ID:** BC30641  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `ByVal` または `ByRef`を指定し、両方は指定しないでください。
