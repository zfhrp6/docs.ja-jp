# .NET ランタイム インターフェイスを作成できません:  &lt;error&gt;
インターフェイスの作成中、またはインターフェイスのメソッド呼び出しが失敗したときに、内部コンパイラ エラーが発生しました。  
  
 **エラー ID:** BC31024  
  
### このエラーを解決するには  
  
1.  作業内容を保存し、[!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)] を再起動します。  
  
2.  エラーがまだ発生する場合は、[!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)] を再インストールします。  
  
3.  再インストールした後にエラーが続く場合は、マイクロソフト プロダクト サポート サービスに通知してください。
