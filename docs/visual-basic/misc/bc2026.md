# 警告番号 &#39;&lt;number&gt;&#39; (オプション &#39;&lt;optionname&gt;&#39;) は、構成できないか無効です
警告 ID が無効です。 ID は 0 より大きい整数値である必要があります。  
  
 **エラー ID:** BC2026  
  
### このエラーを解決するには  
  
-   表示された警告 ID が有効かどうかを確認します。
