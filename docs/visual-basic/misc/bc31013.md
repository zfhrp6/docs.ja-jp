# DLL ファイル &#39;&lt;filename&gt;&#39; を読み込めません: &lt;error&gt;
このプロジェクトで参照されている DLL を読み込むことができませんでした。  
  
 **エラー ID:** BC31013  
  
### このエラーを解決するには  
  
1.  指定した場所にファイルがあることと、ファイルが別のアプリケーションによってロックされていないことを確認します。
