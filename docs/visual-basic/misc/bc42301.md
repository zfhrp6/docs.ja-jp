# 各言語要素につき 1 つの XML コメント ブロックのみが許可されます
複数の XML コメント ブロックが、言語要素に適用されました。  
  
 **エラー ID:** BC42301  
  
### このエラーを解決するには  
  
-   余分な XML コメント ブロックを削除します。
