# <a name="39handles39-in-modules-must-specify-a-39withevents39-variable-qualified-with-a-single-identifier"></a>' ハンドルのモジュールでは、単一の識別子で修飾 'WithEvents' 変数を指定する必要があります
イベント ハンドラーを指定するには、 `Handles` キーワードで宣言されたオブジェクト変数を `WithEvents` ステートメントに指定する必要があります。  
  
 **エラー ID:** BC31418  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `WithEvents` 修飾子を使用して、 `Handles` ステートメントで使用する変数を宣言します。
