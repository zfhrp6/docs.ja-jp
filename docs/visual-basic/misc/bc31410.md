# 複数の基本インターフェイスで宣言されているメソッドのオーバーロードは、有効ではありません
継承された複数のインターフェイスが、同じメソッドを暗黙的にオーバーロードしています。  
  
 **エラー ID:** BC31410  
  
### このエラーを解決するには  
  
-   `Overloads` 修飾子の代わりに `Shadows` 修飾子を使用します。
