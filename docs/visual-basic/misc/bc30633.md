# <a name="39set39-statement-must-end-with-a-matching-39end-set39"></a>&#39;設定&#39;、対応するステートメントが終了する必要があります&#39;最後の設定&#39;
`Set` プロパティ プロシージャは、必ず `End Set` ステートメントで終了する必要があります。  
  
 **エラー ID:** BC30633  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `Set` プロパティ プロシージャが `End Set` コンストラクトで終了していることを確認します。
