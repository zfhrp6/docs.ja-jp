# <a name="white-space-may-not-appear-here"></a>空白はここでは使用できません
XML 要素リテラルの名前に空白が含まれています。  
  
 **エラー ID:** BC31177  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   XML 要素リテラルの名前から空白を削除します。
