# <a name="39lttypenamegt39-is-an-unsupported-type"></a>&#39;です。&lt;typename&gt;&#39; はサポートされていない型です。
指定された型は [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)]で表すことができません。  
  
 **エラー ID:** BC30649  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   サポートされる型を使用します。  
  
-   新しい型を開発します。
