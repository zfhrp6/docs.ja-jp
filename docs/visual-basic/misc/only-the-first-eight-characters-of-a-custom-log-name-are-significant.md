# カスタム ログ名の最初の 8 文字が有効です
イベント ログ名の一意性を調べるときには、最初の 8 文字だけが考慮されます。 最初の 8 文字に同じ文字を使用するイベント ログが複数ある場合は、競合する可能性があります。  
  
### このエラーを解決するには  
  
-   最初の 8 文字が一意になる名前をイベント ログに付けます。
