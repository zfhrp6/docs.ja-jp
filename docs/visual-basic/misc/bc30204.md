# 整数の定数が必要です
2 番目の引数が整数リテラルでない `#ExternalSource` ディレクティブがあります。 このコンテキストでは、整数リテラルだけが有効です。 名前付き定数または列挙型のメンバーは無効です。  
  
 **エラー ID:** BC30204  
  
### このエラーを解決するには  
  
1.  リテラルの代わりに、名前付き定数か列挙体のメンバーを使用します。  
  
2.  `#ExternalSource` ディレクティブの 2 番目の引数として整数リテラルを指定します。
