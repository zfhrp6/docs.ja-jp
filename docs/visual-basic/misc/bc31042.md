# &#39;Sub New&#39; はインターフェイス メンバーを実装できません
`Sub New` はコンストラクターであり、メンバーを実装することはできません。  
  
 **エラー ID:** BC31042  
  
### このエラーを解決するには  
  
-   `Sub New` プロシージャから `Implements` ステートメントを削除します。
