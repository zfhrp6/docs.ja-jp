## <a name="to-correct-this-error"></a><span data-ttu-id="b8e23-106">このエラーを解決するには</span><span class="sxs-lookup"><span data-stu-id="b8e23-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="b8e23-107">インスタンス メンバーのコピー、または `Me` への参照から返される値のコピーを作成し、それをクエリ式の中で使用します。次に例を示します。</span><span class="sxs-lookup"><span data-stu-id="b8e23-107">Create a copy of the instance member or the value returned by the reference to `Me` and use the copy in the query expression, as shown in the following example.</span></span>

    Public Sub SetSearchValue(ByVal number As Integer)

    End Sub
