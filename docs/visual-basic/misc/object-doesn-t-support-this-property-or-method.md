# オブジェクトは、このプロパティまたはメソッドをサポートしていません (Visual Basic)
このオートメーション オブジェクトには存在しないメソッドまたはプロパティを指定しました。  
  
### このエラーを解決するには  
  
-   オブジェクトのドキュメントでオブジェクトの詳細を参照し、プロパティとメソッドのスペルを確認します。
