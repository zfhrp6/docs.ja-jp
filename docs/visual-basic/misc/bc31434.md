# <a name="method-39ltmethodname1gt39-cannot-implement-partial-method-39ltmethodname2gt39-because-39ltmethodname3gt39-already-implements-it"></a>メソッド&#39; &lt;methodname1&gt; &#39;部分メソッドを実装できません&#39; &lt;methodname2&gt; &#39;ため&#39; &lt;methodname3&gt; &#39; 。既に実装しています
メソッド '\<methodname1 >' 部分メソッドを実装できません。'\<methodname2 >' ため '\<methodname3 >' 既に実装しています。 部分メソッドを実装できるのは、1 つのメソッドのみです。  
  
 同じ部分メソッドの宣言を実装する 2 つの部分メソッドがあってはなりません。 次のコードを使用すると、このエラーが発生します。  
  
```vb  
Partial Class Product  
  
    ' Declaration of the partial method.  
    Partial Private Sub ValueChanged()  
    End Sub  
  
End Class  
```  
  
```vb  
Partial Class Product  
  
    ' First implementation of the partial method.  
    Private Sub ValueChanged()  
        MsgBox(Value was changed to " & Me.Quantity)  
    End Sub  
  
    ' Second implementation of the partial method causes this error.  
    'Private Sub ValueChanged()  
    '    Console.WriteLine("Quantity was changed to " & Me.Quantity)  
    'End Sub  
  
End Class  
```  
  
 **エラー ID:** BC31434
