# &#39;&lt;typename&gt;&#39; は &#39;MustInherit&#39; と宣言されているため、属性として使用できません。
カスタム属性クラスは `MustInherit` として宣言できません。  
  
 **エラー ID:** BC31506  
  
### このエラーを解決するには  
  
-   カスタム属性から `MustInherit` 修飾子を削除します。
