# 明示的な型を含む宣言では、型文字 &#39;&lt;character&gt;&#39; を使用できません
データ型を明示的に指定する際に、`$` などの型文字が使用されています。  
  
 **エラー ID:** BC30302  
  
### このエラーを解決するには  
  
-   宣言から型文字を削除するか、`As <Type>` 句を削除します。
