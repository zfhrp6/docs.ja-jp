# インターフェイスは別のインターフェイスからのみ継承することができます
インターフェイス以外のものから継承しようとしました。  
  
 **エラー ID:** BC30354  
  
### このエラーを解決するには  
  
-   基本インターフェイス、または継承しようとしている元のインターフェイスが実際にインターフェイスであることを確認します。
