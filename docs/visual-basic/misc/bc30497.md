# &#39;Sub New&#39; でイベントを処理することはできません
`Sub New` と `Handles` を結合しようとしていますが、この処理は実行できません。 プロシージャ宣言の最後で `Handles` キーワードを使用すると、`WithEvents` キーワードで宣言されたオブジェクト変数によって発生したイベントが処理されるようになります。  
  
 **エラー ID:** BC30497  
  
### このエラーを解決するには  
  
1.  このコンテキストでは `New` を使用しないでください。
