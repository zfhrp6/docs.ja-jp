# <a name="39end-removehandler39-must-be-preceded-by-a-matching-39removehandler39-declaration"></a>&#39;End RemoveHandler&#39;前に、対応する必要がある&#39;RemoveHandler&#39;宣言
`End RemoveHandler` ステートメントが発生しましたが、対応する `RemoveHandler` ステートメントがありません。 `End RemoveHandler` の前に、対応する `RemoveHandler` ステートメントが必要です。  
  
 **エラー ID:** BC31125  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   先行する `RemoveHandler` ステートメントが有効であり、スペルが正しいことを確認します。
