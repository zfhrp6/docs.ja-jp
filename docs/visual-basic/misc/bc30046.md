# メソッドに ParamArray と Optional パラメーターの両方を指定することはできません
`ParamArray` 引数は自動的に省略可能であり、プロシージャ宣言内で唯一の省略可能な引数である必要があります。 先行する引数はすべて必須である必要があります。  
  
 **エラー ID:** BC30046  
  
### このエラーを解決するには  
  
-   宣言内の省略可能な引数を削除します。
