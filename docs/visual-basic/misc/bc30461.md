# <a name="classes-cannot-be-declared-39ltspecifiergt39"></a>クラスを宣言することはできません&#39;&lt;指定子&gt;&#39;
`Class` または `Module` を、無効な指定子を使用して宣言しようとしました。  
  
 **エラー ID:** BC30461  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定子を削除します。
