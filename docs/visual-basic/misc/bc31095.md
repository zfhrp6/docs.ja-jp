# 別のコンストラクターを呼び出している間は、作成中のオブジェクトへの参照は有効ではありません
オブジェクトのコンストラクターによるオブジェクトの作成が完了する前に、そのオブジェクトのメンバーが参照されました。  
  
 **エラー ID:** BC31095  
  
### このエラーを解決するには  
  
-   コンストラクターから別のコンストラクターを呼び出す際には、`MyBase`、`MyClass`、または `Me` を使用しないでください。
