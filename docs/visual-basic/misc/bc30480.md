# <a name="shared-39sub-new39-cannot-be-declared-39ltspecifiergt39"></a>共有&#39;Sub New&#39;として宣言できません&#39;&lt;指定子。&gt;&#39;
`Shared Sub New` 宣言で有効ではない指定子を使用しようとしました。  
  
 **エラー ID:** BC30480  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定子を削除します。
