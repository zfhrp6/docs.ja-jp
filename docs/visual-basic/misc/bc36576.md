# 前にピリオドが付いた識別子が必要です。
プロパティ名を推論できない値が、プロパティ名に割り当てられることなく、匿名型の宣言の初期化子リストに含まれています。  
  
```  
' Not valid. ' Dim anon1 = New With {.grade = 100, 95}  
```  
  
 **エラー ID:** BC36576  
  
### このエラーを解決するには  
  
-   次のコードに示すように、初期化子リスト内の各値にプロパティ名を指定します。  
  
    ```  
    Dim anon2 = New With {.grade1 = 100, .grade2 = 95}  
    ```
