# 名前空間宣言は &#39;xmlns&#39; で始まらなければなりません。
XML 名前空間が、必要な `xmlns` 識別子を使用せずに指定されています。`xmlns` 識別子には、小文字のみを含める必要があります。  
  
 **エラー ID:** BC31187  
  
### このエラーを解決するには  
  
-   XML 名前空間を宣言する場合は、`xmlns` 識別子を使用します。 例を次に示します。  
  
    ```vb#  
    Imports <xmlns:ns="http://SampleNamespace">  
    ```
