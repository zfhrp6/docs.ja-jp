# 別のコンストラクターを呼び出している間は、作成中のオブジェクトへの暗黙的な参照は有効ではありません
オブジェクトのコンストラクターによるオブジェクトの構築が完了する前に、そのオブジェクトのメンバーが参照されました。  
  
 **エラー ID:** BC31096  
  
### このエラーを解決するには  
  
-   コンストラクターから別のコンストラクターを呼び出す際には、`MyBase`、`MyClass`、または `Me` を使用しないでください。
