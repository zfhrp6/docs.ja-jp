# 引数 &#39;&lt;argumentname&gt;&#39; の型は &#39;&lt;typename&gt;&#39; で、数値ではありません
数値引数が期待される状況で数値以外の引数が指定されています。  
  
### このエラーを解決するには  
  
-   引数が数値であることを確認します。
