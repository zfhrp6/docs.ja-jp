# <a name="39error39-statements-are-not-valid-in-the-immediate-window"></a>&#39;エラー&#39;ステートメントは、イミディ エイト ウィンドウでは有効ではありません。
`Error` ステートメントは、ソース コードでのみ使用できます。  
  
 **エラー ID:** BC30724  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   デバッグ コードから `Error` ステートメントを削除します。
