# <a name="39addhandler39-declaration-must-end-with-a-matching-39end-addhandler39"></a>&#39;AddHandler&#39;宣言は、対応する終了する必要があります&#39;End AddHandler&#39;
`AddHandler` 宣言は、 `End AddHandler` ステートメントで終了する必要があります。  
  
 **エラー ID:** BC31115  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `AddHandler` 宣言が `End AddHandler` ステートメントで終了していることを確認します。
