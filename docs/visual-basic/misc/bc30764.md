# 対象となる .NET Compact Framework のバージョンが、遅延バインディングされたオーバーロードの解決をサポートしていません
対象となる .NET Compact Framework のバージョンが、遅延バインディングされたオーバーロードの解決をサポートしていません。  
  
 **エラー ID:** BC30764  
  
### このエラーを解決するには  
  
-   型オブジェクトのパラメーターを適切な型にキャストします。
