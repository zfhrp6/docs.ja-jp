# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-latebound-overload-resolution"></a>対象となる .NET Compact Framework のバージョンが、遅延バインディングされたオーバーロードの解決をサポートしていません
対象となる .NET Compact Framework のバージョンが、遅延バインディングされたオーバーロードの解決をサポートしていません。  
  
 **エラー ID:** BC30764  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   型オブジェクトのパラメーターを適切な型にキャストします。
