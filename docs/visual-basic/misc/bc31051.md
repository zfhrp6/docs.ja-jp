# <a name="namespace-or-type-39ltnamegt39-has-already-been-imported"></a>Namespace または型&#39;&lt;名前&gt;&#39;既にインポートされています
名前空間を複数回インポートしようとしました。  
  
 **エラー ID:** BC31051  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   不要な `Imports` ステートメントを削除します。
