# 配列変数を宣言するために &#39;ReDim&#39; ステートメントを使用することはできません
`ReDim` は既存の配列のサイズを変更するためにのみ使用できます。  
  
 **エラー ID:** BC30811  
  
### このエラーを解決するには  
  
-   宣言されるときに、配列のサイズを指定します。例を示します。  
  
    ```  
    Dim X(20) As Integer  
    ```
