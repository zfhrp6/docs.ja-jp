# <a name="39raiseevent39-declaration-must-end-with-a-matching-39end-raiseevent39"></a>&#39;です。RaiseEvent &#39;宣言の終わりに対応する &#39; にする必要があります。End RaiseEvent &#39;
`RaiseEvent` 宣言は、 `End RaiseEvent` ステートメントで終了する必要があります。  
  
 **エラー ID:** BC31117  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `RaiseEvent` 宣言が `End RaiseEvent` ステートメントで終了することを確認します。
