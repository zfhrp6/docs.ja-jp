# <a name="39with39-contexts-and-statements-are-not-valid-in-debug-windows"></a>&#39;です。&#39;です。コンテキストおよびステートメントには使えないデバッグ ウィンドウです。
`With` ステートメントは、ソース コードでのみ使用できます。  
  
 **エラー ID:** BC30726  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   デバッグ コードから `With` ステートメントを削除します。
