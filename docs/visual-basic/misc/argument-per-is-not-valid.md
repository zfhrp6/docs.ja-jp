# 引数 &#39;Per&#39; は使用できません
`Per` に指定された引数は、1 ～ `NPer` の範囲で支払い期間を指定する `Double` である必要があるため、使用できません。  
  
### このエラーを解決するには  
  
-   引数が正しい範囲内にあることを確認します。
