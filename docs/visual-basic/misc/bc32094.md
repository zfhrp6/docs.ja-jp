# コンマまたは &#39;)&#39; (型引数リスト) が必要です
型引数リストに予期しない要素が含まれています。 コンマ \(`,`\) を使用してこのようなリスト内の型引数を分離し、閉じかっこ \(`)`\) を使用してリストを終了します。  
  
 **エラー ID:** BC32094  
  
### このエラーを解決するには  
  
-   コンマを追加して型引数リストを続けるか、または閉じかっこ文字を追加してリストを終了します。
