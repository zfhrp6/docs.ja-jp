# &#39;&lt;type&gt;&#39; を 2 回以上継承することはできません
`Inherits` ステートメントが重複しています。  
  
 **エラー ID:** BC30584  
  
### このエラーを解決するには  
  
-   不要な `Inherits` ステートメントを削除します。
