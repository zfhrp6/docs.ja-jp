# 停止の要求は保留中です
Visual Studio デバッガーで、プロシージャ呼び出しが式に指定されていますが、スレッド停止の要求があります。  
  
 デバッガーはアクティブでないスレッド上のプロシージャを呼び出そうとしません。  
  
 **エラー ID:** BC30946  
  
### このエラーを解決するには  
  
-   可能であれば、スレッド中断の要求元を調べて、再度要求が行われないようにします。  
  
-   デバッグを終了し、再起動します。
