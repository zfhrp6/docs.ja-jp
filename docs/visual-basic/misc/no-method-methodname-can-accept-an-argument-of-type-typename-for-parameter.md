# メソッド &#39;&lt;methodname&gt;&#39; ではパラメーター &#39;&lt;parametername&gt;&#39; に対して型 &#39;&lt;typename&gt;&#39; の引数を使用できません
メソッドの呼び出しに、無効な型の引数が含まれています。  
  
### このエラーを解決するには  
  
1.  引数の型をご確認ください。  
  
2.  引数の型をメソッドのパラメーター型と照合し、両者が対応することを確かめます。 必要に応じて、型を変更します。
