# <a name="39is39-expected"></a>&#39;&#39;が必要です
`TypeOf` 演算子に `Is` 句がありません。  
  
 **エラー ID:** BC30224  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `Is` 演算子の後に `TypeOf` 句を追加します。たとえば、 `TypeOf MyControl Is ComboBox`のように記述します。
