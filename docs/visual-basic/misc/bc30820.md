# &#39;&lt;lsetstmt&gt;&#39; は宣言されていません
'\<lsetstmt\>' が宣言されていません。 LSet ステートメントは現在サポートされていません。Microsoft.VisualBasic.LSet を使用してください。  
  
 以前のバージョンの [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] に固有のいくつかの関数は <xref:Microsoft.VisualBasic?displayProperty=fullName> 名前空間に移動されています。 これにより、これらの機能をより一般的にすべてのプログラミング言語で使用できます。  
  
 **エラー ID:** BC30820  
  
### このエラーを解決するには  
  
-   `Microsoft.VisualBasic` 名前空間の `LSet` 関数を代わりに使用します。
