# 既定のメンバー &#39;&lt;membername&gt;&#39; はプロパティではありません
`Default` メンバーを宣言しようとしました。 しかし、`Default` キーワードは既定のプロパティを識別するために使用されます。  
  
 **エラー ID:** BC30555  
  
### このエラーを解決するには  
  
-   `Default` を使用してプロパティを識別しようとしていることを確認するか、`Default` キーワードを削除します。
