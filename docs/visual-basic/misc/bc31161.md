# <a name="expected-closing-39--gt39-for-xml-comment"></a>終わり&#39;--&gt; &#39; XML コメント用
XML コメント リテラルの終わりに必要な '-->' 文字がありません。  
  
 **エラー ID:** BC31161  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   XML コメント リテラルの終わりに必要な '-->' 文字を追加します。
