# <a name="39systemruntimeinteropservicesdllimportattribute39-cannot-be-applied-to-a-39declare39"></a>&#39;System.Runtime.InteropServices.DllImportAttribute&#39;を適用することはできません、 &#39;Declare&#39;
`DllImportAttribute` 属性が `Declare` 関数に適用されました。 この属性は、空の `Function` または `Sub`でのみ使用できます。  
  
 **エラー ID:** BC31523  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  `DllImportAttribute` ステートメントから `Declare` 属性を削除します。
