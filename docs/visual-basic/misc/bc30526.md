# プロパティ &#39;&lt;propertyname&gt;&#39; は &#39;ReadOnly&#39; です
`ReadOnly` で宣言されているプロパティに値を代入しようとしました。  
  
 **エラー ID:** BC30526  
  
### このエラーを解決するには  
  
-   プロパティ宣言から `ReadOnly` 指定子を削除します。
