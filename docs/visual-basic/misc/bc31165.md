# <a name="expected-beginning-39lt39-for-an-xml-tag"></a>予期される先頭 &#39;&lt;&#39; XML タグ
XML リテラルの XML タグの先頭に必要な '<' 文字がありません。  
  
 **エラー ID:** BC31165  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   XML リテラルの XML タグの先頭に '<' 文字を追加します。
