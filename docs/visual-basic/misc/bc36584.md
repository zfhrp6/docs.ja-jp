# <a name="parameter-39ltparameternamegt39-in-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39-already-has-a-matching-argument"></a>パラメーター &#39; &lt;parametername&gt; &#39;拡張メソッドで&#39; &lt;methodname&gt; &#39;で定義されている&#39; &lt;typename&gt; &#39;既に一致する引数を持つ
1 つの引数が、同じプロシージャ呼び出しで複数回にわたって名前で渡されています。  
  
 **エラー ID:** BC36584  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   引数名のいずれかのスペルが誤っていないかどうかを確認します。  
  
-   引数リストから重複している引数を削除します。
