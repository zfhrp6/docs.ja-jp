# <a name="39nothing39-cannot-be-evaluated"></a>&#39;です。何も &#39;評価できません。
`Nothing` キーワードがデバッグ ウィンドウで使用されました。  
  
 **エラー ID:** BC30701  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   デバッグ中に `Nothing` を使用しないでください。
