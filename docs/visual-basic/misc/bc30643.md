# <a name="property-39ltpropertynamegt39-is-of-an-unsupported-type"></a>プロパティ&#39; &lt;propertyname&gt; &#39;はサポートされていない型
参照先アセンブリのプロパティには、表現できない型が含まれています。  
  
 **エラー ID:** BC30643  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   アプリケーションで使用できる型を指定します。
