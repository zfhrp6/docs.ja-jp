# <a name="modules-cannot-be-declared-39ltspecifiergt39"></a>モジュールを宣言できません&#39;&lt;指定子。&gt;&#39;
モジュールが、無効なアクセス修飾子を使用して宣言されました。  
  
 **エラー ID:** BC31052  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   アクセス修飾子を `Public` や `Friend`などの有効なアクセス修飾子に置き換えます。
