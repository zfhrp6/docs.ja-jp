# <a name="default-values-cannot-be-supplied-for-parameters-that-are-not-declared-39optional39"></a>既定値は、宣言されていないパラメーターに対して指定することはできません&#39;オプション&#39;
プロシージャを呼び出す場合は必要な引数を指定する必要があり、引数は宣言で既定値を受け取りません。  
  
 **エラー ID:** BC32024  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   引数が `Optional` になるように宣言するか、または既定値を削除します。
