# <a name="argument-39path39-is-nothing-or-empty"></a>引数&#39;パス&#39;は Nothing または空にします。
パス名は、 `Nothing` に初期化されるか空であるため、使用できません。 パスには、特定のデータ型と正しい名前を設定する必要があります。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   正しいパス名を追加します。
