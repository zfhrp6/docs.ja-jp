# <a name="39ltassemblynamegt39-is-an-assembly-and-cannot-be-referenced-as-a-module"></a>&#39;&lt;assemblyname&gt; &#39;アセンブリ、モジュールとして参照できません。
アセンブリをモジュールとして使用しようとしました。  
  
 **エラー ID:** BC31077  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   モジュールへの参照を変更します。
