# &#39;In&#39; が必要です
`From` または `Aggregate` 句が指定されていますが、`In` 演算子がありません。`In` 演算子を使用して、クエリを実行するコレクションを識別します。  
  
 **エラー ID:** BC36607  
  
### このエラーを解決するには  
  
1.  `From` または `Aggregate` 句に `In` 演算子とキー フィールドを追加します。 次に例を示します。  
  
    ```vb#  
    Dim names = From pers In people Select pers.FirstName, pers.LastName  
    ```
