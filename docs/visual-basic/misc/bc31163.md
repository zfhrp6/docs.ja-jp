# XML 属性値の終わりに、対応する単一引用符が必要です
単一引用符で区切られた属性値に、終わりの引用符文字がありません。  
  
 **エラー ID:** BC31163  
  
### このエラーを解決するには  
  
-   属性値を区切るための終わりの単一引用符文字 \('\) を追加します。
