# &#39;End&#39; ステートメントは、イミディエイト ウィンドウでは有効ではありません
`Stop` と `End` のステートメントは実行を中断するため、デバッグ コンテキストでは使用できません。  
  
 **エラー ID:** BC30123  
  
### このエラーを解決するには  
  
-   **\[イミディエイト\]** ウィンドウで `End` または `Stop` のステートメントを実行しないようにします。
