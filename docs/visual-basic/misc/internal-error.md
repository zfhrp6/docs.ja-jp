# 内部エラー (Visual Basic)
[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] で内部の不具合が発生しました。  
  
### このエラーを解決するには  
  
-   このエラーが `Error` ステートメントまたは `Raise` メソッドによって生成されたのではないことを確認します。 そのうえで、マイクロソフト製品サポート サービスにお問い合わせいただき、メッセージが表示されている条件をご報告ください。
