# &#39;Resume&#39; ステートメントは、イミディエイト ウィンドウでは有効ではありません
Resume ステートメントは、ソース コードでのみ使用できます。  
  
 **エラー ID:** BC30714  
  
### このエラーを解決するには  
  
-   デバッグ コードから `Resume` ステートメントを削除します。
