# <a name="unable-to-open-response-file-39ltfilenamegt39"></a>応答ファイルを開けません&#39;&lt;ファイル名。&gt;&#39;
アクセスしようとしているファイルは使用不可です。  
  
 **エラー ID:** BC2011  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   ネットワーク接続がアクティブであることを確認してください。  
  
-   ファイルのアクセス許可を確認します。
