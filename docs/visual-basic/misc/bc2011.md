# <a name="unable-to-open-response-file-39ltfilenamegt39"></a>応答ファイル &#39; を開けません。&lt;filename&gt;&#39;です。
アクセスしようとしているファイルは使用不可です。  
  
 **エラー ID:** BC2011  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   ネットワーク接続がアクティブであることを確認してください。  
  
-   ファイルのアクセス許可を確認します。
