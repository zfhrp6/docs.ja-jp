# 応答ファイル &#39;&lt;filename&gt;&#39; を開くことができません。
アクセスしようとしているファイルは使用不可です。  
  
 **エラー ID:** BC2011  
  
### このエラーを解決するには  
  
-   ネットワーク接続がアクティブであることを確認してください。  
  
-   ファイルのアクセス許可を確認します。
