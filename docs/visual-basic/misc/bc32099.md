# コンマまたは &#39;)&#39; が必要です (型パラメーター リスト)
型パラメーター リストに予期しない要素が含まれています。 コンマ \(`,`\) を使用してこのようなリスト内の型パラメーターを分離し、終わりかっこ \(`)`\) を使用してリストを終了します。  
  
 **エラー ID:** BC32099  
  
### このエラーを解決するには  
  
-   コンマを追加して型パラメーター リストを続けるか、または終わりかっこ文字を追加してリストを終了します。
