# &#39;Finally&#39; からの分岐は有効ではありません
`Finally` ブロック内の `GoTo` ステートメントが、ブロック外に分岐しています。`Catch` ブロックまたは `Finally` ブロックの中へ、またはそれらのブロックから外へ分岐するのは正しくありません。  
  
 **エラー ID:** BC30101  
  
### このエラーを解決するには  
  
-   `GoTo` ステートメントを削除し、条件判断構造またはループ制御構造を使用するプログラム ロジックの実装をご検討ください。
