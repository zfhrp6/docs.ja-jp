# &#39;System.Runtime.InteropServices.DllImportAttribute&#39; を &#39;Get&#39; または &#39;Set&#39; に適用することはできません
`DllImportAttribute` 属性が `Get` または `Set` プロパティ プロシージャに適用されました。  
  
 **エラー ID:** BC31524  
  
### このエラーを解決するには  
  
1.  `Get` および `Set` プロパティ プロシージャから `DllImportAttribute` を削除します。
