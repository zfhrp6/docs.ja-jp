# <a name="39implements39-statement-must-follow-any-39inherits39-statement-and-precede-all-declarations-in-a-class"></a>&#39; implements &#39;ステートメントがいずれかの &#39; Inherits &#39; に従う必要があります。ステートメントでクラスのすべての宣言の前に、
`Implements` ステートメントを無効な場所に記述しました。  
  
 **エラー ID:** BC31053  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `Implements` ステートメントを `Inherits` ステートメントの直後かつどの宣言よりも前に配置します。 例:  
  
    ```  
    Class Derived  
       Inherits Base  
       Implements One  
       Sub SubOne() Implements One.Method1  
          ' Add code to implement the method.  
       End Sub  
    End Class  
    ```
