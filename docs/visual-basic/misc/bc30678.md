# &#39;End&#39; ステートメントが有効ではありません。
`End` キーワードが正しくない方法で使用されているか、または現在のコンテキストで適切ではありません。  
  
 **エラー ID:** BC30678  
  
### このエラーを解決するには  
  
1.  `End` を単独で使用して、プロシージャがスコープを失うようにします。  
  
2.  `End` その他のキーワードと一緒に使用して、構造体やプロパティなどのコードのブロックを終了します。
