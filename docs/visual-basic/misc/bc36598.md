# <a name="instance-of-restricted-type-39lttypenamegt39-cannot-be-used-in-a-query-expression"></a>制限付きの型のインスタンス&#39; &lt;typename&gt; &#39;クエリ式では使用できません
<xref:System.ArgIterator>、 <xref:System.RuntimeArgumentHandle>、 <xref:System.TypedReference>など制限付きの型のインスタンスをラムダ式またはクエリ式で使用することはできません。  
  
 **エラー ID:** BC36598
