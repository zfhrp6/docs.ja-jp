# &#39;MyClass&#39; をクラスの外で使用することはできません
クラス構造の外部で `MyClass` キーワードを使用しました。  
  
 **エラー ID:** BC30470  
  
### このエラーを解決するには  
  
-   `MyClass` 宣言をクラス構造の中に移動します。
