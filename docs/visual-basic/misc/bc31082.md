# <a name="39ltvariablenamegt39-is-not-a-local-variable-or-parameter-and-so-cannot-be-used-as-a-39catch39-variable"></a>&#39;&lt;variablename&gt; &#39;ローカル変数またはパラメーターではないととして使用することはできません、&#39;キャッチ&#39;変数
`Try...Catch...Finally` ステートメント内の変数は、ローカルで宣言されているか、現在のプロシージャのパラメーターである必要があります。  
  
 **エラー ID:** BC31082  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  `Try...Catch...Finally` ステートメントのローカル変数またはパラメーターを宣言します。
