# 配列の配列を初期化する場合、トップレベル配列に対してのみ境界を指定できます
ジャグ配列 \(配列の配列\) の配列初期化では、いずれかの下位レベルの初期の長さが設定されます。 配列の宣言ステートメントでは、最上位の配列のみの長さを指定できます。  
  
 **エラー ID:** BC32014  
  
### このエラーを解決するには  
  
1.  最上位の配列を除くすべての配列から長さの指定を削除します。  
  
2.  後続の代入ステートメントで `New` キーワードを使用して、下位レベルの配列の初期の長さを設定します。
