# 定数式の先頭で &#39;.&#39; または &#39;!&#39; を使用することはできません
メンバー アクセス \(.\) およびディクショナリ メンバー アクセス \(\!\) では、ほとんどの場合、メンバが含まれている要素を指定した式が必要になります。定数式の場合もそうです。 次の宣言は無効です。  
  
```  
' Not valid. Const c As String = .name  
```  
  
 **エラー ID:** BC30995  
  
### このエラーを解決するには  
  
-   アクセスするメンバーが含まれているインスタンスを指定します。
