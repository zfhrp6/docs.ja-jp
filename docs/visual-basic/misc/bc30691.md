# <a name="39lttypenamegt39-is-a-type-in-39ltcontainernamegt39-and-cannot-be-used-as-an-expression"></a>&#39;&lt;typename&gt; &#39;内の型は、 &#39; &lt;containername&gt; &#39; 、式として使用できません
使用中の名前が型の名前として既に宣言されているため、式では使用できません。  
  
 **エラー ID:** BC30691  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   メンバーに一意の名前を付けます。  
  
-   メンバーの名前を角かっこで囲みます。
