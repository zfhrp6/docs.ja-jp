# &#39;&lt;typename&gt;&#39; は &#39;&lt;containername&gt;&#39; では型です。式として使用することはできません。
使用中の名前が型の名前として既に宣言されているため、式では使用できません。  
  
 **エラー ID:** BC30691  
  
### このエラーを解決するには  
  
-   メンバーに一意の名前を付けます。  
  
-   メンバーの名前を角かっこで囲みます。
