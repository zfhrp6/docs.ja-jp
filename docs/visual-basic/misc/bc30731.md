# &#39;AddressOf&#39; 式は、デバッグ ウィンドウ内では無効です
`AddressOf` ステートメントは、ソース コードでのみ使用できます。  
  
 **エラー ID:** BC30731  
  
### このエラーを解決するには  
  
1.  デバッグ コードから `AddressOf` ステートメントを削除します。
