# <a name="no-accessible-overloaded-39ltmethodnamegt39-can-be-called-with-these-arguments-without-a-widening-conversion-ltlistgt"></a>ないオーバー ロードされた&#39; &lt;methodname&gt; &#39;拡大変換しないで、これらの引数と共に呼び出せる:&lt;一覧&gt;
オーバーロードされたメソッドが呼び出されましたが、拡大変換せずに指定された引数のリストと一致するメソッドはありませんでした。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `Option Strict Off`を指定します。  
  
-   オーバーロードされたメソッドのシグネチャのいずれかと一致するように、引数を変更します。
