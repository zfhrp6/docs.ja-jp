# 指定されたレジストリ キーが存在しません
指定されたレジストリ キーが存在しません。  
  
### このエラーを解決するには  
  
-   指定したレジストリ キーを調べて、パスを正しく指定していることを確認してください。  
  
-   書き込みを試行する前にレジストリ キーを作成します。
