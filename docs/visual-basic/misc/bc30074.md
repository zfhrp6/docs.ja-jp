# 定数を代入式のターゲットにすることはできません
値を代入するコンテキストに定数があります。 実行時に値を代入できるのは、書き込み可能な変数、プロパティ、配列要素だけです。  
  
 **エラー ID:** BC30074  
  
### このエラーを解決するには  
  
-   定数を、書き込み可能な単独の変数、プロパティ、または配列要素に置き換えます。
