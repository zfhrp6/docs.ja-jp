# <a name="39ltqualifiedmethodgt39-is-not-accessible-in-this-context-because-it-is-39ltmodifiergt39"></a>&#39;&lt;qualifiedmethod&gt; &#39;になっているためにはこのコンテキストでアクセスできなくなって&#39;&lt;修飾子&gt;&#39;
修飾子が適用されているためにこのコンテキストでアクセスできないメソッドにアクセスしようとしました。  
  
 **エラー ID:** BC30390  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   修飾子を削除します。
