# Const 宣言に、配列初期化子を指定することはできません
`Const` 宣言で配列初期化子を使用しようとしました。  
  
 **エラー ID:** BC30445  
  
### このエラーを解決するには  
  
-   かっこを削除することで、初期化子を有効な式に変更します。
