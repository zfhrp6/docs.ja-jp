<span data-ttu-id="3d1cd-103">非ジェネリック型を呼び出す際に、宣言ステートメントまたは代入ステートメントに [Of](../../visual-basic/language-reference/statements/of-clause.md) 句が含まれています。</span><span class="sxs-lookup"><span data-stu-id="3d1cd-103">A declaration or assignment statement includes an [Of](../../visual-basic/language-reference/statements/of-clause.md) clause when invoking a non-generic type.</span></span>  
  
 <span data-ttu-id="3d1cd-104">その定義によって、 *ジェネリック型* は、クラス、構造体、インターフェイス、プロシージャ、またはデリゲートになり、1 つ以上の *型パラメーター*で指定できるデータ型で動作します。</span><span class="sxs-lookup"><span data-stu-id="3d1cd-104">By its definition, a *generic type* is a class, structure, interface, procedure, or delegate that operates on data types you can specify through one or more *type parameters*.</span></span> <span data-ttu-id="3d1cd-105">コードを使用してこのジェネリック型から型を作成すると、型パラメーターそれぞれに *型引数* が指定されます。</span><span class="sxs-lookup"><span data-stu-id="3d1cd-105">When the using code creates a type from this generic type, it supplies a *type argument* to each type parameter.</span></span> <span data-ttu-id="3d1cd-106">型の作成の一環として、型引数はそれぞれ、生成されたコード内にある対応する型パラメーターの出現個所をすべて置換します。</span><span class="sxs-lookup"><span data-stu-id="3d1cd-106">As part of the creation of the type, each type argument replaces every occurrence of its corresponding type parameter in the generated code.</span></span>  
  
 <span data-ttu-id="3d1cd-107">型パラメーターはかっこ内の `Of` 句を使用して定義され、型引数はかっこ内の `Of` 句を使用して指定されます。</span><span class="sxs-lookup"><span data-stu-id="3d1cd-107">Type parameters are defined with an `Of` clause inside parentheses, and type arguments are supplied using an `Of` clause inside parentheses.</span></span> <span data-ttu-id="3d1cd-108">`Of` 句は、ジェネリック型を処理する場合にのみ使用されます。</span><span class="sxs-lookup"><span data-stu-id="3d1cd-108">The `Of` clause is used only when dealing with generic types.</span></span>  
  
 <span data-ttu-id="3d1cd-109">非ジェネリック型は型パラメーターを受け入れません。このような型を呼び出す場合は、型引数を指定できません。</span><span class="sxs-lookup"><span data-stu-id="3d1cd-109">Non-generic types do not accept type parameters, and you cannot specify any type arguments when you invoke such a type.</span></span>
