# <a name="goto-to-label-39ltlabelnamegt39-cannot-be-used-within-lambda-expressions"></a>GoTo ラベル &#39;&lt;labelname&gt;&#39; ラムダ式内で使用することはできません
ラムダ式の本体は、計算を実行して単一の値を返す式を指定する必要があります。 `GoTo` ステートメントを含む、すべてのステートメントが許可されません。  
  
 **エラー ID:** BC36597
