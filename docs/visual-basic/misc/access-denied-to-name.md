# <a name="access-denied-to-39ltnamegt39"></a>アクセス拒否 &#39;&lt;名前&gt;&#39;です。
コンピューターにおけるセキュリティ アクセス許可が不十分であるため、デバイス、ファイル、ディスクにアクセスできません。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  必要なアクセス許可があることを確認します。
