# 引数 &lt;argumentname&gt; は、0 以上である必要があります
引数の値が 0 未満です。 引数の値は、0 以上である必要があります。  
  
### このエラーを解決するには  
  
1.  引数の値が 0 以上であることを確認します。  
  
2.  引数が計算によって指定される場合は、計算で処理される値を確認します。
