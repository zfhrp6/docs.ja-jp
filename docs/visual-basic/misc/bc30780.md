# <a name="39continue39-statements-are-not-valid-in-the-immediate-window"></a>&#39;続行&#39;ステートメントは、イミディ エイト ウィンドウでは有効ではありません。
`Continue` ステートメントは、ソース コードでのみ使用できます。  
  
 **エラー ID:** BC30780  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `Continue` ステートメントをデバッグ コードから削除します。
