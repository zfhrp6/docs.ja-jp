# 構造体の中のメソッドを、&#39;Protected&#39; または &#39;Protected Friend&#39; に宣言することはできません
構造体の中のメソッドが `Protected` または `Protected Friend` と宣言されました。  
  
 **エラー ID:** BC31067  
  
### このエラーを解決するには  
  
-   構造体内部のメソッドを `Public` または `Private` と宣言します。
