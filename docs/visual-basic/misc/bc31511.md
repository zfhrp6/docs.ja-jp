# 属性メンバー &#39;&lt;membername&gt;&#39; は &#39;Public&#39; と宣言されていないため、代入式のターゲットにすることはできません。
属性のプライベート メンバーに値を代入しようとしました。  
  
 **エラー ID:** BC31511  
  
### このエラーを解決するには  
  
1.  代入を削除します。  
  
2.  開発したカスタム属性を使用する場合は、属性メンバーのアクセス修飾子を `Public` に変更します。
