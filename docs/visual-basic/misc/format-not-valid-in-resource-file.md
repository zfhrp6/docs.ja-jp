# 形式がリソース ファイルで有効ではありません
プロジェクト内のリソース ファイルが Windows リソース ファイルの標準ファイル形式に準拠していないか、リソース ファイルと [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] プロジェクトの間で不一致があります。  
  
### このエラーを解決するには  
  
1.  リソース ソースとリソース ファイルを作成し直し、Windows Software Development Kit に記載されている構文に準拠します。  
  
2.  無効なリソース ファイルをプロジェクトから削除します。
