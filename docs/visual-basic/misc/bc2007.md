# <a name="unrecognized-option-ltoptionnamegt-ignored"></a>認識できないオプション&lt;optionname&gt;。 無視されます。
コマンド ラインから無効なオプションが指定されています。  
  
 **エラー ID:** BC2007  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   無効なオプションがないか、指定したオプションを確認します。
