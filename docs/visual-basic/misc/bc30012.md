# <a name="39if39-block-must-end-with-a-matching-39end-if39"></a>&#39;#If&#39; 、対応するブロックを終了する必要があります&#39;#End If&#39;
`#If` は条件付きコンパイル ディレクティブです。 `#If` ブロックが `#End If` ディレクティブで終了していません。  
  
 **エラー ID:** BC30012  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   条件付きコンパイル ブロックの最後に `#End If` ディレクティブ追加します。
