# <a name="39set39-is-already-declared"></a>&#39;設定&#39;は既に宣言されています
コード ブロックで `Set` ステートメントが複数回出現します。 `Set` ステートメントは、値をプロパティに割り当てるのに使用する `Set` プロパティ プロシージャを宣言します。  
  
 **エラー ID:** BC30444  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   不要な `Set` ステートメントを削除します。
