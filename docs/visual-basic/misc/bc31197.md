# 全角文字は、XML の区切り記号としては無効です
区切り記号として全角文字を含む、XML リテラルが定義されています。 全角文字はワイド文字またはマルチバイト文字とも呼ばれます。  
  
 **エラー ID:** BC31197  
  
### このエラーを解決するには  
  
-   XML リテラルの定義から全角文字を削除し、有効な ANSI 区切り文字に置き換えます。 有効な区切り記号の文字には `<`、`>`、`=`、`:`、`/` があります。
