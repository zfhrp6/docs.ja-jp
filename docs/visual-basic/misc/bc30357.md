# インターフェイス内では、&#39;Inherits&#39; ステートメントはすべての宣言の前に記述しなければなりません
`Inherits` ステートメントは、クラス定義の最初の空行でもコメント行でもない行に記述されている必要があります。  
  
 **エラー ID:** BC30357  
  
### このエラーを解決するには  
  
-   すべての宣言の前に来るように `Inherits` ステートメントを移動します。
