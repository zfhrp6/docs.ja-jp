# <a name="39end-module39-must-be-preceded-by-a-matching-39module39"></a>&#39;End Module&#39;前に、対応する必要がある&#39;モジュール&#39;
モジュール ブロックは `Module` ステートメントで始まり、 `End Module` コンストラクトで終わります。  
  
 **エラー ID:** BC30622  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   モジュール ブロックが `Module` ステートメントで始まることを確認してください。
