# My.Application.Log は空きディスク領域の容量を求めることができません
<xref:Microsoft.VisualBasic.Logging.FileLogTraceListener> クラスが空きディスク領域の容量を求めることができませんでした。  
  
### このエラーを解決するには  
  
-   エラーが発生した状況を記録して、マイクロソフト プロダクト サポート サービスにご連絡ください。
