# &#39;WithEvents&#39; 変数には &#39;As&#39; 句を指定しなければなりません
`As` 句に `WithEvents` キーワードを指定していません。 イベントを発生可能な特定のクラスとして、変数を宣言します。  
  
 **エラー ID:** BC30412  
  
### このエラーを解決するには  
  
1.  イベントで発生可能なクラスを指定する `As` 句を追加します。
