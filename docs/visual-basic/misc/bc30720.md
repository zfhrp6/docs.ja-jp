# &#39;On Error&#39; ステートメントは、イミディエイト ウィンドウでは有効ではありません
`On Error` ステートメントは、ソース コードでのみ使用できます。  
  
 **エラー ID:** BC30720  
  
### このエラーを解決するには  
  
-   デバッグ コードから `On Error` ステートメントを削除します。
