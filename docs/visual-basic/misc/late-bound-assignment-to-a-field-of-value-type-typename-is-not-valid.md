# &#39;&lt;name&gt;&#39; が遅延バインディングの結果である場合、&#39;&lt;typename&gt;&#39; 型の値のフィールドへの遅延バインディングの割り当ては無効です。
無効な遅延バインディングの割り当てが試行されました。  
  
### このエラーを解決するには  
  
-   事前バインディングされた割り当てを行います。
