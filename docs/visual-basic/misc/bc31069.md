# インターフェイス内の Enum を &#39;&lt;specifier&gt;&#39; と宣言することはできません
インターフェイスで宣言された列挙体で、無効な修飾子が指定されました。  
  
 **エラー ID:** BC31069  
  
### このエラーを解決するには  
  
-   `Public`、`Private`、`Shared`、`Friend`、`Protected`、`Protected Friend`、または `Overridable` などの修飾子を削除します。
