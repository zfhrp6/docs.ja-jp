# 構造体で、共有されていない、パラメーターなしの &#39;Sub New&#39; を宣言することはできません
構造体の中で宣言された `Sub New` コンストラクターは、引数を受け入れるか、または `Shared` 修飾子で宣言する必要があります。  
  
 **エラー ID:** BC30629  
  
### このエラーを解決するには  
  
-   引数を受け入れるように `Sub New` コンストラクターを変更します。  
  
-   `Shared` 修飾子を適用して、コンストラクターを共有します。
