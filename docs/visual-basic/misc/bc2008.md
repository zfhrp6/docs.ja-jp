# <a name="no-input-sources-specified"></a>入力ソースが指定されていません。
コンパイル時に、入力ソースが指定されていません。  
  
 **エラー ID:** BC2008  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   ソース コード ファイルが含まれていることを確認します。
