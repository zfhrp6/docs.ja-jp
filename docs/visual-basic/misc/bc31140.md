# &#39;Custom&#39; 修飾子は &#39;Event&#39; 宣言の直前にのみ使用することができます。
カスタム イベントの宣言で `Custom` キーワードは `Event` キーワードの前にしか置くことができません。  
  
 **エラー ID:** BC31140  
  
### このエラーを解決するには  
  
-   `Custom` キーワードを `Event` キーワードの前に移動させます。  
  
     または  
  
-   `Custom` キーワードを削除します。
