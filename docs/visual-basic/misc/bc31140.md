# <a name="39custom39-modifier-can-only-be-used-immediately-before-an-39event39-declaration"></a>'Custom' 修飾子は、イベントの宣言の直前にのみ使用できます。
カスタム イベントの宣言で `Custom` キーワードは `Event` キーワードの前にしか置くことができません。  
  
 **エラー ID:** BC31140  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `Custom` キーワードを `Event` キーワードの前に移動させます。  
  
     または  
  
-   `Custom` キーワードを削除します。
