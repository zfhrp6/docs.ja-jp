# 引数 &#39;&lt;argumentname&gt;&#39; は多次元配列にはなることはできません
配列引数のランクが 1 を超えていますが、この配列には 1 次元配列のみを指定できます。  
  
### このエラーを解決するには  
  
-   引数のランクを 1 に変更します。
