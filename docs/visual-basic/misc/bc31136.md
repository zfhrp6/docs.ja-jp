# <a name="39addhandler39-and-39removehandler39-method-parameters-must-have-the-same-delegate-type-as-the-containing-event"></a>&#39;AddHandler&#39;と&#39;RemoveHandler&#39;メソッドのパラメーターは、として同じデリゲート型を含むイベントを持つ必要があります
`Custom Event` 宣言には、 `AddHandler` 宣言または `RemoveHandler` 宣言が必要です。それぞれの宣言は、カスタム イベントの `As` 句で指定されたデリゲート型の 1 つのパラメーターを取ります。  
  
 **エラー ID:** BC31136  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   パラメーターの型が、カスタム イベント `As` 句で指定されたデリゲート型と同じになるように変更します。
