# ステートメントは、メソッド本体の内部には記述できません
プロシージャに、別のプロシージャ宣言などの無効なステートメントが含まれています。 プロシージャ宣言を入れ子にすることはできません。  
  
 **エラー ID:** BC30289  
  
### このエラーを解決するには  
  
-   プロシージャ宣言を削除するか、または外側のプロシージャの外に移動します。
