# <a name="39inherits39-statement-must-precede-all-declarations-in-a-class"></a>&#39; 継承 &#39;です。ステートメントは、クラス内のすべての宣言を付ける必要があります。
`Inherits` ステートメントは、クラス定義の先頭にある必要があります。  
  
 **エラー ID:** BC30683  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `Inherits` ステートメントをクラスの先頭に移動します。
