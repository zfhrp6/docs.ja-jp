# &#39;Inherits&#39; ステートメントはクラス内のすべての宣言の前に記述する必要があります
`Inherits` ステートメントは、クラス定義の先頭にある必要があります。  
  
 **エラー ID:** BC30683  
  
### このエラーを解決するには  
  
-   `Inherits` ステートメントをクラスの先頭に移動します。
