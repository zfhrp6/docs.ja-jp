# <a name="expected-closing-3939-for-xml-entity"></a>予想クローズ&#39;です。&#39; XML エンティティの
XML エンティティ参照 (例: `>`など) で最後のセミコロン (;) が見つかりません。  
  
 **エラー ID:** BC31178  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   XML エンティティ参照の末尾にセミコロンを含めます。
