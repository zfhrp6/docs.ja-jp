# <a name="compiler-error-cs0558"></a><span data-ttu-id="73e7e-102">コンパイラ エラー CS0558</span><span class="sxs-lookup"><span data-stu-id="73e7e-102">Compiler Error CS0558</span></span>
<span data-ttu-id="73e7e-103">ユーザー定義の演算子 'operator' は static および public として宣言されなければなりません</span><span class="sxs-lookup"><span data-stu-id="73e7e-103">User-defined operator 'operator' must be declared static and public</span></span>  
  
 <span data-ttu-id="73e7e-104">ユーザー定義の演算子では、 **static** と **public** の両方のアクセス [修飾子](../../csharp/language-reference/keywords/modifiers.md) を指定する必要があります。</span><span class="sxs-lookup"><span data-stu-id="73e7e-104">Both the **static** and **public** access [modifiers](../../csharp/language-reference/keywords/modifiers.md) must be specified on user-defined operators.</span></span>  
  
 <span data-ttu-id="73e7e-105">次の例では CS0558 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="73e7e-105">The following sample generates CS0558:</span></span>  
  
```csharp  
// CS0558.cs  
namespace x  
{  
   public class ii  
   {  
      public class iii  
      {  
         static implicit operator int(iii aa)   // CS0558, add public  
         {  
            return 0;  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
