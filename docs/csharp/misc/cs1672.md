# <a name="compiler-error-cs1672"></a>コンパイラ エラー CS1672
/platform に対するオプション 'option' が正しくありません。anycpu、x86、Itanium、または x64 を指定してください  
  
 このオプションはプロセッサの種類を指定します。リストされている形式のいずれかを使用します。
