 <span data-ttu-id="f68e4-104">非プロパティ データ型を [プロパティ](../../csharp/programming-guide/classes-and-structs/properties.md)としてオーバーライドしようとしましたが、許可されません。</span><span class="sxs-lookup"><span data-stu-id="f68e4-104">An attempt was made to override a nonproperty data type as a [property](../../csharp/programming-guide/classes-and-structs/properties.md), which is not allowed.</span></span>

```csharp

   public int i;  
}

      {  
         return 0;  
      }  
   }  
}  
```
