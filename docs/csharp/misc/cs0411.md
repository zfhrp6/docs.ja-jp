## <a name="example"></a><span data-ttu-id="43a8f-107">例</span><span class="sxs-lookup"><span data-stu-id="43a8f-107">Example</span></span>  
 <span data-ttu-id="43a8f-108">次の例では CS0411 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="43a8f-108">The following sample generates CS0411:</span></span>  
  
```csharp

{  
    void G<T>()  
    {  
    }  
  
    public static void Main()  
    {  
        G();  // CS0411  
        // Try this instead:  
        // G<int>();  
    }  
}  
```
