# <a name="compiler-error-cs0155"></a><span data-ttu-id="72cdb-102">コンパイラ エラー CS0155</span><span class="sxs-lookup"><span data-stu-id="72cdb-102">Compiler Error CS0155</span></span>
<span data-ttu-id="72cdb-103">キャッチ、または スローされた型は System.Exception から派生したものでなければなりません。</span><span class="sxs-lookup"><span data-stu-id="72cdb-103">The type caught or thrown must be derived from System.Exception</span></span>  
  
 <span data-ttu-id="72cdb-104">派生していないデータ型を渡すが試み**System.Exception**に、[キャッチ](../../csharp/language-reference/keywords/try-catch.md)ブロックします。</span><span class="sxs-lookup"><span data-stu-id="72cdb-104">An attempt was made to pass a data type that does not derive from **System.Exception** into a [catch](../../csharp/language-reference/keywords/try-catch.md) block.</span></span> <span data-ttu-id="72cdb-105">**System.Exception** から派生したデータ型のみ **catch** ブロックに渡すことができます。</span><span class="sxs-lookup"><span data-stu-id="72cdb-105">Only data types that derive from **System.Exception** can be passed into a **catch** block.</span></span> <span data-ttu-id="72cdb-106">詳細については、次を参照してください。[例外処理ステートメント](../../csharp/language-reference/keywords/exception-handling-statements.md)と[例外と例外処理](../../csharp/programming-guide/exceptions/index.md)します。</span><span class="sxs-lookup"><span data-stu-id="72cdb-106">For more information, see [Exception Handling Statements](../../csharp/language-reference/keywords/exception-handling-statements.md) and [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="72cdb-107">次の例では CS0155 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="72cdb-107">The following sample generates CS0155:</span></span>  
  
```  
// CS0155.cs  
using System;  
  
namespace MyNamespace  
{  
    public class MyClass2  
    // try the following line instead  
    // public class MyClass2 : Exception  
    {  
    }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            try  
            {  
            }  
            catch (MyClass2)   // CS0155, resolves if you derive MyClass2 from Exception  
            {  
            }  
        }  
    }  
}  
```
