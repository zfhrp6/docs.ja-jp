# <a name="compiler-error-cs0023"></a>コンパイラ エラー CS0023
演算子 'operator' を 'type' 型のオペランドに適用することはできません  
  
 変数に演算子を適用しようとしましたが、変数の型は演算子に対して機能するように設計されていません。 詳細については、次を参照してください。[型](../../csharp/programming-guide/types/index.md)と[c# 演算子](../../csharp/language-reference/operators/index.md)です。  
  
 次の例では CS0023 が生成されます。  
  
```csharp  
// CS0023.cs  
namespace x  
{  
   public class a  
   {  
      public static void Main()  
      {  
         string s = "hello";  
         s = -s;   // CS0023, minus operator not allowed on strings  
      }  
   }  
}  
```
