## <a name="example"></a><span data-ttu-id="06d19-107">例</span><span class="sxs-lookup"><span data-stu-id="06d19-107">Example</span></span>  
 <span data-ttu-id="06d19-108">次の例では、CS3010 エラーが生成されます。</span><span class="sxs-lookup"><span data-stu-id="06d19-108">The following example generates CS3010:</span></span>  
  
```  
// CS3010.cs  
  
using System;  
  
[assembly:CLSCompliant(true)]  
public interface I  
{  
    [CLSCompliant(false)]  
    int M();   // CS3010  
}  
  
public class C : I  
{  
    public int M()  
    {  
        return 1;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
