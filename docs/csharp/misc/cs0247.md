# コンパイラ エラー CS0247
stackalloc で負のサイズを使うことはできません。  
  
 [stackalloc](../../csharp/language-reference/keywords/stackalloc.md) ステートメントに負の値が渡されました。  
  
 次の例では CS0247 が生成されます。  
  
```  
// CS0247.cs // compile with: /unsafe public class MyClass { unsafe public static void Main() { int *p = stackalloc int [-30];   // CS0247 } }  
```
