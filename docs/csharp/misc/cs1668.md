# <a name="compiler-warning-level-2-cs1668"></a>コンパイラの警告 (レベル 2) CS1668
正しくない検索パス 'path' が 'path string' で指定されました -- 'system error message'  
  
 コマンド ラインで [/lib](../../csharp/language-reference/compiler-options/lib-compiler-option.md) に指定されたパスが正しくないか、LIB 環境変数のパスが正しくありません。 パスが存在しアクセスできることを確認するために、使用されるパスをチェックします。 単一引用符内のエラー メッセージは、オペレーティング システムから返されたエラーです。
