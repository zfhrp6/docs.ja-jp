# <a name="compiler-error-cs1555"></a>コンパイラ エラー CS1555
Main メソッドに指定された 'class' が見つかりませんでした  
  
 [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) コンパイラ オプションにクラスが指定されましたが、クラス名がソース コードに見つかりませんでした。
