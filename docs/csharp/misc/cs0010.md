# <a name="compiler-error-cs0010"></a>コンパイラ エラー CS0010
予期しない致命的なエラーです。'error'  
  
 まったく予期しないことが発生すると、このエラーが生成され、コンパイルが停止します。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  別のプロジェクトを再コンパイルします。 同じエラーが発生した場合は、Visual Studio を再インストールしてください。 このエラーが発生した場合は、エラー報告を Microsoft に送信してください。
