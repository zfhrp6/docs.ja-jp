# <a name="compiler-warning-level-1-cs0809"></a>コンパイラの警告 (レベル 1) CS0809
旧形式のメンバー 'memberA' が、旧形式でないメンバー 'memberB' をオーバーライドします。  
  
 通常、不使用とマークされているメンバーで不使用とマークされていないメンバーをオーバーライドしないでください。 この警告が生成される[!INCLUDE[vs_orcas_long](../../csharp/misc/includes/vs_orcas_long_md.md)]にはない[!INCLUDE[vsprvslong](../../csharp/misc/includes/vsprvslong_md.md)]します。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  `Obsolete` 属性を、オーバーライドするメンバーから削除するか、または基底クラスのメンバーに追加します。  
  
## <a name="example"></a>例  
  
```  
// CS0809.cs  
public class Base  
{  
    public virtual void Test1()  
    {  
    }  
}  
public class C : Base  
{  
    [System.Obsolete()]  
    public override void Test1() // CS0809  
    {  
    }  
}  
```
