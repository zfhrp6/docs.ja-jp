# <a name="compiler-warning-level-2-cs0252"></a><span data-ttu-id="2c038-102">コンパイラの警告 (レベル 2) CS0252</span><span class="sxs-lookup"><span data-stu-id="2c038-102">Compiler Warning (level 2) CS0252</span></span>
<span data-ttu-id="2c038-103">予期しない参照比較です。比較値を取得するには、型 'type' に左辺をキャストしてください</span><span class="sxs-lookup"><span data-stu-id="2c038-103">Possible unintended reference comparison; to get a value comparison, cast the left hand side to type 'type'</span></span>  
  
 <span data-ttu-id="2c038-104">コンパイラは参照比較を行っています。</span><span class="sxs-lookup"><span data-stu-id="2c038-104">The compiler is doing a reference comparison.</span></span> <span data-ttu-id="2c038-105">文字列の値を比較する場合は、式の左辺を `type`にキャストします。</span><span class="sxs-lookup"><span data-stu-id="2c038-105">If you want to compare the value of strings, cast the left side of the expression to `type`.</span></span>  
  
 <span data-ttu-id="2c038-106">次の例では CS0252 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="2c038-106">The following sample generates CS0252:</span></span>  
  
```csharp  
// CS0252.cs  
// compile with: /W:2  
using System;  
  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool b = o == s;   // CS0252  
      // try the following line instead  
      // bool b = (string)o == s;  
   }  
}  
```
