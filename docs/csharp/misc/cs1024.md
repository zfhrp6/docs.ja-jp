# コンパイラ エラー CS1024
プリプロセッサ ディレクティブが必要です。  
  
 行の先頭はシャープ記号 \(\#\) ですが、後続の文字列が有効な[プリプロセッサ ディレクティブ](../../visual-basic/reference/command-line-compiler/index.md)ではありませんでした。  
  
 次の例では CS1024 が生成されます。  
  
```  
// CS1024.cs #import System   // CS1024  
```
