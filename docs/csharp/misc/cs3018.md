 <span data-ttu-id="e20a4-104">この警告は、CLSCompliant 属性が `true` に設定された入れ子のクラスが、CLSCompliant 属性が `false`に設定されたクラスのメンバーとして宣言されている場合に発生します。</span><span class="sxs-lookup"><span data-stu-id="e20a4-104">This warning occurs if a nested class with the CLSCompliant attribute set to `true` is declared as a member of a class declared with the CLSCompliant attribute set to `false`.</span></span> <span data-ttu-id="e20a4-105">CLS 準拠ではない外部クラスのメンバーである場合、入れ子になったクラスを CLS 準拠にすることはできないため、このような記述は使用できません。</span><span class="sxs-lookup"><span data-stu-id="e20a4-105">This is not allowed, since a nested class cannot be CLS-compliant if it is a member of an outer class that is not CLS-compliant.</span></span> <span data-ttu-id="e20a4-106">この警告を解決するには、入れ子になったクラスから CLSCompliant 属性を削除するか、属性の設定を `true` から `false`に変更します。</span><span class="sxs-lookup"><span data-stu-id="e20a4-106">To resolve this warning, remove the CLSCompliant attribute from the nested class, or change it from `true` to `false`.</span></span> <span data-ttu-id="e20a4-107">CLS 準拠について詳しくは、「[CLS 準拠コードの記述](http://msdn.microsoft.com/library/4c705105-69a2-4e5e-b24e-0633bc32c7f3)」および「[言語への非依存性、および言語非依存コンポーネント](../../../docs/standard/language-independence-and-language-independent-components.md)」をご覧ください。</span><span class="sxs-lookup"><span data-stu-id="e20a4-107">For more information on CLS Compliance, see [Writing CLS-Compliant Code](http://msdn.microsoft.com/library/4c705105-69a2-4e5e-b24e-0633bc32c7f3) and [Language Independence and Language-Independent Components](../../../docs/standard/language-independence-and-language-independent-components.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="e20a4-108">例</span><span class="sxs-lookup"><span data-stu-id="e20a4-108">Example</span></span>  
 <span data-ttu-id="e20a4-109">次の例では CS3018 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="e20a4-109">The following sample generates CS3018.</span></span>  
  
```csharp  
// CS3018.cs  
// compile with: /target:library  
using System;  
  
[assembly: CLSCompliant(true)]  
[CLSCompliant(false)]  
public class Outer  
{  
   [CLSCompliant(true)]   // CS3018  
   public class Nested {}  
  
   // OK  
   public class Nested2 {}  
  
   [CLSCompliant(false)]  
   public class Nested3 {}  
}  
```
