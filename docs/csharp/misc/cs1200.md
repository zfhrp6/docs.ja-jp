# <a name="compiler-warning-level-1-cs1200"></a>コンパイラの警告 (レベル 1) CS1200
機能 'invalid feature' は推奨されません。 代わりに 'valid feature' を使用してください。  
  
 使用しようとした機能は、現在使用されていません。 代わりに、正しい機能を使用するようにコードを更新します。
