# <a name="compiler-warning-level-1-cs3001"></a>コンパイラの警告 (レベル 1) CS3001
引数型 'type' は CLS に準拠していません  
  
 A[パブリック](../../csharp/language-reference/keywords/public.md)、[保護](../../csharp/language-reference/keywords/protected.md)、または`protected internal`メソッドの型が共通言語仕様 (CLS) に準拠しているパラメーターを受け入れる必要があります。 CLS 準拠について詳しくは、「[CLS 準拠コードの記述](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3)」および「[言語への非依存性、および言語非依存コンポーネント](https://msdn.microsoft.com/library/12a7a7h3)」をご覧ください。  
  
## <a name="example"></a>例  
 次の例では、CS3001 エラーが生成されます。  
  
```  
// CS3001.cs  
  
[assembly:System.CLSCompliant(true)]  
public class a  
{  
    public void bad(ushort i)   // CS3001  
    {  
    }  
  
    private void OK(ushort i)   // OK, private method  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
