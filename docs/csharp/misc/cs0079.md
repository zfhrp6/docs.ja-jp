# <a name="compiler-error-cs0079"></a><span data-ttu-id="c70d4-102">コンパイラ エラー CS0079</span><span class="sxs-lookup"><span data-stu-id="c70d4-102">Compiler Error CS0079</span></span>
<span data-ttu-id="c70d4-103">イベント 'event' は += または -= の左側でのみ使用できます</span><span class="sxs-lookup"><span data-stu-id="c70d4-103">The event 'event' can only appear on the left hand side of += or -=</span></span>  
  
 <span data-ttu-id="c70d4-104">[event](../../csharp/language-reference/keywords/event.md) が正しく呼び出されませんでした。</span><span class="sxs-lookup"><span data-stu-id="c70d4-104">An [event](../../csharp/language-reference/keywords/event.md) was called incorrectly.</span></span> <span data-ttu-id="c70d4-105">詳細については、次を参照してください。[イベント](../../csharp/programming-guide/events/index.md)と[デリゲート](../../csharp/programming-guide/delegates/index.md)です。</span><span class="sxs-lookup"><span data-stu-id="c70d4-105">For more information, see [Events](../../csharp/programming-guide/events/index.md) and [Delegates](../../csharp/programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="c70d4-106">次の例では CS0079 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="c70d4-106">The following sample generates CS0079:</span></span>  
  
```csharp  
// CS0079.cs  
using System;  
  
public delegate void MyEventHandler();  
  
public class Class1  
{  
   private MyEventHandler _e;  
  
   public event MyEventHandler Pow  
   {  
      add  
      {  
         _e += value;  
         Console.WriteLine("in add accessor");  
      }  
      remove  
      {  
         _e -= value;  
         Console.WriteLine("in remove accessor");  
      }  
   }  
  
   public void Handler()  
   {  
   }  
  
   public void Fire()  
   {  
      if (_e != null)  
      {  
         Pow();   // CS0079  
         // try the following line instead  
         // _e();  
      }  
   }  
  
   public static void Main()  
   {  
      Class1 p = new Class1();  
      p.Pow += new MyEventHandler(p.Handler);  
      p._e();  
      p.Pow += new MyEventHandler(p.Handler);  
      p._e();  
      p._e -= new MyEventHandler(p.Handler);  
      if (p._e != null)  
      {  
         p._e();  
      }  
      p.Pow -= new MyEventHandler(p.Handler);  
      if (p._e != null)  
      {  
         p._e();  
      }  
   }  
}  
```
