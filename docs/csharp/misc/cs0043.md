# <a name="compiler-error-cs0043"></a>コンパイラ エラー CS0043
PDB ファイル 'file' の形式が正しくないか、古い形式です。 削除してリビルドしてください。  
  
 このコンパイルの .pdb ファイルを削除して再コンパイルします。
