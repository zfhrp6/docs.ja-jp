# コンパイラの警告 (レベル 1) CS1592
組み込みコメント ファイルの中の XML 形式が正しくありません \-\- 'reason'  
  
 ***reason***  として報告された問題が、[\<include\>](../../csharp/programming-guide/xmldoc/include.md) タグで指定されたファイルに含まれています。
