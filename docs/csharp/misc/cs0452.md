<span data-ttu-id="4f852-103">型 'type name' は、ジェネリック型のパラメーター 'parameter name'、またはメソッド 'identifier of generic' として使用するために、参照型でなければなりません</span><span class="sxs-lookup"><span data-stu-id="4f852-103">The type 'type name' must be a reference type in order to use it as parameter 'parameter name' in the generic type or method 'identifier of generic'</span></span>
