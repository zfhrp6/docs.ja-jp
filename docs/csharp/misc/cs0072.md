# <a name="compiler-error-cs0072"></a><span data-ttu-id="23fed-102">コンパイラ エラー CS0072</span><span class="sxs-lookup"><span data-stu-id="23fed-102">Compiler Error CS0072</span></span>
<span data-ttu-id="23fed-103">'event' : オーバーライドできません。'method' はイベントではありません</span><span class="sxs-lookup"><span data-stu-id="23fed-103">'event' : cannot override; 'method' is not an event</span></span>  
  
 <span data-ttu-id="23fed-104">[イベント](../../csharp/language-reference/keywords/event.md) は別のイベントのみをオーバーライドできます。</span><span class="sxs-lookup"><span data-stu-id="23fed-104">An [event](../../csharp/language-reference/keywords/event.md) can only override another event.</span></span> <span data-ttu-id="23fed-105">詳細については、「[イベント](../../csharp/programming-guide/events/index.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="23fed-105">For more information, see [Events](../../csharp/programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="23fed-106">次の例では CS0072 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="23fed-106">The following sample generates CS0072:</span></span>  
  
```  
// CS0072.cs  
delegate void MyDelegate();  
  
class Test1  
{  
   public virtual event MyDelegate MyEvent;  
   public virtual void VMeth()  
   {  
   }  
  
   public void FireAway()  
   {  
      if (MyEvent != null)  
         MyEvent();  
   }  
}  
  
class Test2 : Test1  
{  
   public override event MyDelegate VMeth   // CS0072  
   // uncomment the following lines to resolve  
   // public override event MyDelegate MyEvent  
   {  
      add  
      {  
         VMeth += value;  
         // MyEvent += value;  
      }  
      remove  
      {  
         VMeth -= value;  
         // MyEvent -= value;  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
