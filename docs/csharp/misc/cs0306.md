# <a name="compiler-error-cs0306"></a>コンパイラ エラー CS0306
型 'type' は型引数に使用できません  
  
 型パラメーターとして使用された型は許可されていません。 型がポインター型である可能性があります。  
  
 次の例では CS0306 が生成されます。  
  
```csharp  
// CS0306.cs  
class C<T>  
{  
}  
  
class M  
{  
    // CS0306 – int* not allowed as a type parameter  
     C<int*> f;  
}  
```
