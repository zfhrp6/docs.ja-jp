 <span data-ttu-id="90ffe-104">メンバーは、 [シール](../../csharp/language-reference/keywords/override.md) されている継承されたメンバーを [オーバーライド](../../csharp/language-reference/keywords/sealed.md) することはできません。</span><span class="sxs-lookup"><span data-stu-id="90ffe-104">A member cannot [override](../../csharp/language-reference/keywords/override.md) a [sealed](../../csharp/language-reference/keywords/sealed.md) inherited member.</span></span> <span data-ttu-id="90ffe-105">詳細については、次を参照してください。[オンとオフ](../../csharp/language-reference/keywords/checked-and-unchecked.md)です。</span><span class="sxs-lookup"><span data-stu-id="90ffe-105">For more information, see [Checked and Unchecked](../../csharp/language-reference/keywords/checked-and-unchecked.md).</span></span>  
  
 <span data-ttu-id="90ffe-106">次の例では CS0239 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="90ffe-106">The following sample generates CS0239:</span></span>  
  
```csharp  
// CS0239.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public override sealed void f()  
   {  
   }  
}  
  
class MyClass3 : MyClass2  
{  
   public override void f()   // CS0239  
   // Try the following definition instead:  
   // public new void f()  
   {  
   }  
}  
```
