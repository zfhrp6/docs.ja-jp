# <a name="compiler-error-cs0835"></a><span data-ttu-id="f23b3-102">コンパイラ エラー CS0835</span><span class="sxs-lookup"><span data-stu-id="f23b3-102">Compiler Error CS0835</span></span>
<span data-ttu-id="f23b3-103">ラムダ式を、型引数 'type' がデリゲート型ではない式ツリーに変換できません。</span><span class="sxs-lookup"><span data-stu-id="f23b3-103">Cannot convert lambda to an expression tree whose type argument 'type' is not a delegate type.</span></span>  
  
 <span data-ttu-id="f23b3-104">ラムダ式が式ツリーに変換される場合、式ツリーには、その引数のデリゲート型が必要です。</span><span class="sxs-lookup"><span data-stu-id="f23b3-104">If a lambda expression is converted to an expression tree, the expression tree must have a delegate type for its argument.</span></span> <span data-ttu-id="f23b3-105">さらに、ラムダ式は、そのデリゲート型に変換できる必要があります。</span><span class="sxs-lookup"><span data-stu-id="f23b3-105">Furthermore, the lambda expression must be convertible to the delegate type.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="f23b3-106">このエラーを解決するには</span><span class="sxs-lookup"><span data-stu-id="f23b3-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="f23b3-107">型パラメーターを `int` からデリゲート型 (たとえば、 `Func<int,int>`) に変更します。</span><span class="sxs-lookup"><span data-stu-id="f23b3-107">Change the type parameter from `int` to a delegate type, for example `Func<int,int>`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="f23b3-108">例</span><span class="sxs-lookup"><span data-stu-id="f23b3-108">Example</span></span>  
 <span data-ttu-id="f23b3-109">次の例では CS0835 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="f23b3-109">The following example generates CS0835:</span></span>  
  
```csharp  
// cs0835.cs  
using System;  
using System.Linq;  
using System.Linq.Expressions;  
  
public class C  
{  
    public static int Main()  
    {  
        Expression<int> e = x => x + 1; // CS0835  
  
        // Try the following line instead.  
       // Expression<Func<int,int>> e2 = x => x + 1;  
  
        return 1;  
    }  
}  
```
