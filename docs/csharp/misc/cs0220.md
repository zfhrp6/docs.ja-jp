 <span data-ttu-id="ca859-104">[checked](../../csharp/language-reference/keywords/checked.md)で操作が検出されました。これは既定であり、データの損失が発生しました。</span><span class="sxs-lookup"><span data-stu-id="ca859-104">An operation was detected by [checked](../../csharp/language-reference/keywords/checked.md), which is the default, and a data loss resulted.</span></span> <span data-ttu-id="ca859-105">このエラーを解決するには、代入を修正するか、または [unchecked](../../csharp/language-reference/keywords/unchecked.md) を使用します。</span><span class="sxs-lookup"><span data-stu-id="ca859-105">Either correct the inputs to the assignment or use [unchecked](../../csharp/language-reference/keywords/unchecked.md) to resolve this error.</span></span> <span data-ttu-id="ca859-106">詳細については、次を参照してください。[オンとオフ](../../csharp/language-reference/keywords/checked-and-unchecked.md)です。</span><span class="sxs-lookup"><span data-stu-id="ca859-106">For more information, see [Checked and Unchecked](../../csharp/language-reference/keywords/checked-and-unchecked.md).</span></span>  
  
 <span data-ttu-id="ca859-107">次の例では CS0220 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="ca859-107">The following sample generates CS0220:</span></span>  
  
```  
// CS0220.cs  
using System;  
  
class TestClass  
{  
   const int x = 1000000;  
   const int y = 1000000;  
  
   public int MethodCh()  
   {  
      int z = (x * y);   // CS0220  
      return z;  
   }  
  
   public int MethodUnCh()  
   {  
      unchecked  
      {  
         int z = (x * y);  
         return z;  
      }  
   }  
  
   public static void Main()  
   {  
      TestClass myObject = new TestClass();  
      Console.WriteLine("Checked  : {0}", myObject.MethodCh());  
      Console.WriteLine("Unchecked: {0}", myObject.MethodUnCh());  
   }  
}  
```
