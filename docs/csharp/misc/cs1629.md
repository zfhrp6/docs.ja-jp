 <span data-ttu-id="44e1f-104">C# 言語仕様は、反復子に安全でないコードを許可していません。</span><span class="sxs-lookup"><span data-stu-id="44e1f-104">The C# language specification does not allow unsafe code in iterators.</span></span>  
  
 <span data-ttu-id="44e1f-105">次の例では CS1629 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="44e1f-105">The following sample generates CS1629:</span></span>  
  
```csharp  
// CS1629.cs  
// compile with: /unsafe    
using System.Collections.Generic;  
class C   
{  
   IEnumerator<int> IteratorMeth() {  
      int i;  
      unsafe  // CS1629  
      {  
         int *p = &i;  
         yield return *p;  
      }  
   }  
}  
```
