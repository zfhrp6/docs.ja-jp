# <a name="compiler-error-cs0175"></a><span data-ttu-id="3bfa2-102">コンパイラ エラー CS0175</span><span class="sxs-lookup"><span data-stu-id="3bfa2-102">Compiler Error CS0175</span></span>
<span data-ttu-id="3bfa2-103">キーワード 'base' の使用はこのコンテキストでは有効ではありません。</span><span class="sxs-lookup"><span data-stu-id="3bfa2-103">Use of keyword 'base' is not valid in this context</span></span>  
  
 <span data-ttu-id="3bfa2-104">[base](../../csharp/language-reference/keywords/base.md) キーワードを使用して、基底クラスの特定のメンバーを指定する必要があります。</span><span class="sxs-lookup"><span data-stu-id="3bfa2-104">The [base](../../csharp/language-reference/keywords/base.md) keyword must be used to specify a particular member of the base class.</span></span> <span data-ttu-id="3bfa2-105">詳細については、「[コンストラクター](../../csharp/programming-guide/classes-and-structs/constructors.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="3bfa2-105">For more information, see [Constructors](../../csharp/programming-guide/classes-and-structs/constructors.md).</span></span>  
  
 <span data-ttu-id="3bfa2-106">次の例では CS0175 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="3bfa2-106">The following sample generates CS0175:</span></span>  
  
```  
// CS0175.cs  
using System;  
class BaseClass  
{  
    public int TestInt = 0;  
}  
  
class MyClass : BaseClass  
{  
    public static void Main()  
    {  
        MyClass aClass = new MyClass();  
        aClass.BaseTest();  
    }  
  
    public void BaseTest()  
    {  
        Console.WriteLine(base); // CS0175  
        // Try the following line instead:  
        // Console.WriteLine(base.TestInt);  
        base = 9;   // CS0175  
  
        // Try the following line instead:  
        // base.TestInt = 9;  
    }  
}  
```
