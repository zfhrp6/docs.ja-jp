<span data-ttu-id="baf7f-103">インデクサーに void 型を指定できません。</span><span class="sxs-lookup"><span data-stu-id="baf7f-103">Indexers cannot have void type</span></span>  
  
 <span data-ttu-id="baf7f-104">[インデクサー](../../csharp/programming-guide/indexers/index.md) の戻り値の型を `void`にすることはできません</span><span class="sxs-lookup"><span data-stu-id="baf7f-104">The return type of an [indexer](../../csharp/programming-guide/indexers/index.md) cannot be `void`.</span></span> <span data-ttu-id="baf7f-105">インデクサーは値を返す必要があります。</span><span class="sxs-lookup"><span data-stu-id="baf7f-105">An indexer must return a value.</span></span>  
  
 <span data-ttu-id="baf7f-106">次の例では CS0620 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="baf7f-106">The following sample generates CS0620:</span></span>  
  
```csharp  
// CS0620.cs

{  
   public static void Main()  
   {  
      MyClass test = new MyClass();  
      System.Console.WriteLine(test[2]);  
   }  
  
   void this [int intI]   // CS0620, return type cannot be void

      get  
      {  
         // will need to return some value  
      }  
   }  
}  
```
