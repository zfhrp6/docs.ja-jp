 <span data-ttu-id="913be-108">次の例では CS0307 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="913be-108">The following sample generates CS0307:</span></span>  
  
```csharp  
// CS0307.cs

{  
   public int P { get { return 1; } }  
   public static void Main()  
   {  
      C c = new C();  
      int p = c.P<int>();  // CS0307 – C.P is a property  
      // Try this instead  
      // int p = c.P;  
   }  
}  
```
