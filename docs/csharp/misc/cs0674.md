# <a name="compiler-error-cs0674"></a>コンパイラ エラー CS0674
'System.ParamArrayAttribute' は使用しないでください。 代わりに 'params' キーワードを使用します。  
  
 C# コンパイラがの使用を許可していない<xref:System.ParamArrayAttribute?displayProperty=nameWithType>; を使用して[params](../../csharp/language-reference/keywords/params.md)代わりにします。  
  
 次の例では CS0674 が生成されます。  
  
```csharp  
// CS0674.cs  
using System;  
public class MyClass   
{  
  
   public static void UseParams([ParamArray] int[] list)   // CS0674  
   // try the following line instead  
   // public static void UseParams(params int[] list)   
   {  
      for ( int i = 0 ; i < list.Length ; i++ )  
         Console.WriteLine(list[i]);  
      Console.WriteLine();  
   }  
  
   public static void Main()   
   {  
      UseParams(1, 2, 3);  
   }  
}  
```
