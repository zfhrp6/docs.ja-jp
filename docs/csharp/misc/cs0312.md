## <a name="example"></a><span data-ttu-id="19232-111">例</span><span class="sxs-lookup"><span data-stu-id="19232-111">Example</span></span>

```csharp

class Program

    static void MTyVar<T, U>() where T : U { }  
  
    static int Main()  
    {  
        MTyVar<int?, int>(); // CS0312  
        return 1;  
    }  
}  
```
