# <a name="compiler-error-cs0082"></a><span data-ttu-id="bb977-102">コンパイラ エラー CS0082</span><span class="sxs-lookup"><span data-stu-id="bb977-102">Compiler Error CS0082</span></span>
<span data-ttu-id="bb977-103">型 'type' は、'name' と呼ばれるメンバーを同じパラメーターの型で既に予約しています</span><span class="sxs-lookup"><span data-stu-id="bb977-103">Type 'type' already reserves a member called 'name' with the same parameter types</span></span>  
  
 <span data-ttu-id="bb977-104">コンパイル時に、プロパティは、識別子の前の `get_` と `set_` 、またはそのいずれかでメソッドに変換されます。</span><span class="sxs-lookup"><span data-stu-id="bb977-104">Properties at compile time are translated to methods with `get_` and/or `set_` in front of the identifier.</span></span> <span data-ttu-id="bb977-105">メソッド名と競合する独自のメソッドを定義すると、エラーが生成されます。</span><span class="sxs-lookup"><span data-stu-id="bb977-105">If you define your own method that conflicts with the method name, an error is generated.</span></span>  
  
## <a name="example"></a><span data-ttu-id="bb977-106">例</span><span class="sxs-lookup"><span data-stu-id="bb977-106">Example</span></span>  
 <span data-ttu-id="bb977-107">次の例では CS0082 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="bb977-107">The following example generates CS0082:</span></span>  
  
```csharp  
//cs0082.cs  
class MyClass  
{  
  
    //property  
    public int MyProp  
    {  
        get //CS0082  
        {  
            return 1;  
        }  
    }  
  
    //conflicting Get  
    public int get_MyProp()  
    {  
        return 2;  
    }  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
