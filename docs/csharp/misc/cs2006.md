# <a name="compiler-error-cs2006"></a>コンパイラ エラー CS2006
コマンドライン構文エラー: 'option' オプションの 'text' がありません  
  
 *option* の構文には追加のテキストが必要です。 詳しくは、「 [コンパイラ オプション](../../csharp/language-reference/compiler-options/index.md)」をご覧ください。
