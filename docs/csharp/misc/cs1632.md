# <a name="compiler-error-cs1632"></a><span data-ttu-id="3ae3a-102">コンパイラ エラー CS1632</span><span class="sxs-lookup"><span data-stu-id="3ae3a-102">Compiler Error CS1632</span></span>
<span data-ttu-id="3ae3a-103">コントロールを匿名メソッドまたはラムダ式の本体外に出すことはできません</span><span class="sxs-lookup"><span data-stu-id="3ae3a-103">Control cannot leave the body of an anonymous method or lambda expression</span></span>  
  
 <span data-ttu-id="3ae3a-104">ジャンプ ステートメントにこのエラーが発生した (**break**、 `goto`、**続行**など) を匿名メソッド ブロックからコントロールを移動しようとします。</span><span class="sxs-lookup"><span data-stu-id="3ae3a-104">This error occurs if a jump statement (**break**, `goto`, **continue**, etc.) attempts to move control out of an anonymous method block.</span></span> <span data-ttu-id="3ae3a-105">匿名メソッド ブロックは関数本体であり、return ステートメントまたはブロックの末尾への到達によってのみ終了できます。</span><span class="sxs-lookup"><span data-stu-id="3ae3a-105">An anonymous method block is a function body and can only be exited by a return statement or by reaching the end of the block.</span></span>  
  
 <span data-ttu-id="3ae3a-106">次の例では CS1632 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="3ae3a-106">The following sample generates CS1632:</span></span>  
  
```csharp  
// CS1632.cs  
// compile with: /target:library  
delegate void MyDelegate();  
class MyClass  
{  
   public void Test()  
   {        
      for (int i = 0 ; i < 5 ; i++)  
      {  
         MyDelegate d = delegate {  
            break;   // CS1632  
          };          
      }  
   }  
}  
```
