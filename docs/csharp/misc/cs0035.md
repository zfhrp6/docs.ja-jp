# <a name="compiler-error-cs0035"></a><span data-ttu-id="7a6e9-102">コンパイラ エラー CS0035</span><span class="sxs-lookup"><span data-stu-id="7a6e9-102">Compiler Error CS0035</span></span>
<span data-ttu-id="7a6e9-103">演算子 'operator' は型 'type' のオペランドに対してあいまいです</span><span class="sxs-lookup"><span data-stu-id="7a6e9-103">Operator 'operator' is ambiguous on an operand of type 'type'</span></span>  
  
 <span data-ttu-id="7a6e9-104">コンパイラには複数の使用可能な変換がありますが、どの変換が選択されるかは演算子を適用する前はわかりません。</span><span class="sxs-lookup"><span data-stu-id="7a6e9-104">The compiler has more than one available conversion and does not know which to choose before applying the operator.</span></span> <span data-ttu-id="7a6e9-105">詳細については、次を参照してください。[テンプレート化されたユーザー定義の変換](/cpp/misc/templated-user-defined-conversions)と[変換演算子](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md)です。</span><span class="sxs-lookup"><span data-stu-id="7a6e9-105">For more information, see [Templated User Defined Conversions](/cpp/misc/templated-user-defined-conversions) and [Conversion Operators](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span></span>  
  
 <span data-ttu-id="7a6e9-106">次の例では CS0035 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="7a6e9-106">The following sample generates CS0035:</span></span>  
  
```csharp  
// CS0035.cs  
class MyClass  
{  
   private int i;  
  
   public MyClass(int i)  
   {  
      this.i = i;  
   }  
  
   public static implicit operator double(MyClass x)  
   {  
      return (double) x.i;  
   }  
  
   public static implicit operator decimal(MyClass x)  
   {  
      return (decimal) x.i;  
   }  
}  
  
class MyClass2  
{  
   static void Main()  
   {  
      MyClass x = new MyClass(7);  
      object o = - x;   // CS0035  
      // try a cast:  
      // object o = - (double)x;  
   }  
}  
```
