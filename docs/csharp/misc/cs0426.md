# <a name="compiler-error-cs0426"></a><span data-ttu-id="3f2a7-102">コンパイラ エラー CS0426</span><span class="sxs-lookup"><span data-stu-id="3f2a7-102">Compiler Error CS0426</span></span>
<span data-ttu-id="3f2a7-103">型名 'identifier' は型 'type' に存在しません</span><span class="sxs-lookup"><span data-stu-id="3f2a7-103">The type name 'identifier' does not exist in the type 'type'</span></span>  
  
 <span data-ttu-id="3f2a7-104">このエラーは、別の型の内部に入れ子にされた型を参照したときに、そのような入れ子にされた型が存在しない場合に発生します。</span><span class="sxs-lookup"><span data-stu-id="3f2a7-104">This error occurs when you reference a type nested within another type, but no such nested type exists.</span></span> <span data-ttu-id="3f2a7-105">これは、入れ子にされた型の名前を入力し間違えた場合に発生することがあります。</span><span class="sxs-lookup"><span data-stu-id="3f2a7-105">This can occur if you mistype the name of the nested type.</span></span> <span data-ttu-id="3f2a7-106">使用する名前のスペルをチェックし、外側の型が必要なメンバーを含んでいることを確認します。</span><span class="sxs-lookup"><span data-stu-id="3f2a7-106">Check the spelling of the names used, and verify that the enclosing type has the expected member.</span></span>  
  
 <span data-ttu-id="3f2a7-107">次の例では、クラス C に入れ子にされた型 A が存在しないため、CS0426 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="3f2a7-107">The following sample generates CS0426 because class C has no nested type A:</span></span>  
  
```  
// CS0426.cs  
  
class C  
{  
    // No nested types are declared.     
}  
  
class D  
{  
   public static void Main()  
   {  
       C c = new C();  
       // Attempt to reference a nested type A:  
       C.A a; // CS0426 because no such type C.A  
   }  
}  
```
