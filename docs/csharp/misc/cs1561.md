# <a name="compiler-error-cs1561"></a>コンパイラ エラー CS1561
出力ファイル名が長すぎるか、または無効です。  
  
 出力ファイル名は 256 文字以上にはできません。また、ユーロ記号、疑問符、円記号などの無効な文字を含めることもできません。
