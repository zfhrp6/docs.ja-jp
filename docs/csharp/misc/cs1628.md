# <a name="compiler-error-cs1628"></a><span data-ttu-id="50149-102">コンパイラ エラー CS1628</span><span class="sxs-lookup"><span data-stu-id="50149-102">Compiler Error CS1628</span></span>
<span data-ttu-id="50149-103">Ref または out パラメーター 'parameter' 匿名メソッド、ラムダ式、またはクエリ式の内部に使用することはできません。</span><span class="sxs-lookup"><span data-stu-id="50149-103">Cannot use in ref or out parameter 'parameter' inside an anonymous method, lambda expression, or query expression</span></span>  
  
 <span data-ttu-id="50149-104">使用する場合、このエラーが発生した、 `in`、 `ref`、または`out`匿名メソッド ブロック内のパラメーターです。</span><span class="sxs-lookup"><span data-stu-id="50149-104">This error occurs if you use an `in`, `ref`, or `out` parameter inside an anonymous method block.</span></span> <span data-ttu-id="50149-105">このエラーを回避するには、ローカル変数を使用するか、別のコンストラクトを使用します。</span><span class="sxs-lookup"><span data-stu-id="50149-105">To avoid this error, use a local variable or some other construct.</span></span>  
  
 <span data-ttu-id="50149-106">次の例では CS1628 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="50149-106">The following sample generates CS1628:</span></span>  
  
```csharp  
// CS1628.cs  
  
delegate int MyDelegate();  
  
class C  
{  
  public static void F(ref int i)  
  {  
      MyDelegate d = delegate { return i; };  // CS1628  
      // Try this instead:  
      // int tmp = i;  
      // MyDelegate d = delegate { return tmp; };  
  }  
  
  public static void Main()  
  {  
  
  }  
}  
```
