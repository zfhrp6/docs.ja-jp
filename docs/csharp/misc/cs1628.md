 <span data-ttu-id="bc121-104">このエラーは、ref パラメーターまたは out パラメーターを匿名メソッド ブロックの内側で使用した場合に発生します。</span><span class="sxs-lookup"><span data-stu-id="bc121-104">This error occurs if you use a ref or out parameter inside an anonymous method block.</span></span> <span data-ttu-id="bc121-105">このエラーを回避するには、ローカル変数を使用するか、別のコンストラクトを使用します。</span><span class="sxs-lookup"><span data-stu-id="bc121-105">To avoid this error, use a local variable or some other construct.</span></span>

```  
{  
  public static void F(ref int i)  
  {  
      MyDelegate d = delegate { return i; };  // CS1628  
      // Try this instead:  
      // int tmp = i;  
      // MyDelegate d = delegate { return tmp; };  
  }  
  
  public static void Main()  
  {  
  
  }  
}  
```
