# コンパイラ エラー CS1547
キーワード void はこのコンテキストで使用できません。  
  
 コンパイラは、[void](../../csharp/language-reference/keywords/void.md) キーワードの無効な使用を検出しました。  
  
 次の例では CS1547 が生成されます。  
  
```  
// CS1547.cs public class MyClass { void BadMethod() { void i;   // CS1547, cannot have variables of type void } public static void Main() { } }  
```
