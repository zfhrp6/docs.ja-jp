# <a name="compiler-error-cs0516"></a>コンパイラ エラー CS0516
コンストラクター 'constructor' はそれ自体を呼び出すことはできません  
  
 プログラムでコンストラクターを再帰的に呼び出すことはできません。  
  
 次の例では CS0516 が生成されます。  
  
```csharp  
// CS0516.cs  
namespace x  
{  
   public class clx  
   {  
      public clx() : this()   // CS0516, delete "this()"  
      {  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
