# コンパイラ エラー CS1675
Enums に型パラメーターを指定することはできません。  
  
 このエラーを解決するには、`enum` 宣言から型パラメーターを削除します。  
  
## 使用例  
 次の例では CS1675 が生成されます。  
  
```  
// CS1675.cs enum E<T>  // CS1675 { } class CMain { public static void Main() { } }  
```
