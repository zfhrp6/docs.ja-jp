 <span data-ttu-id="083cf-104">イベントのアクセサー関数を [interface](../../csharp/language-reference/keywords/interface.md)で定義することはできません。</span><span class="sxs-lookup"><span data-stu-id="083cf-104">You cannot define an event's accessor functions in an [interface](../../csharp/language-reference/keywords/interface.md).</span></span> <span data-ttu-id="083cf-105">詳細については、次を参照してください。[イベント](../../csharp/programming-guide/events/index.md)と[インターフェイス](../../csharp/programming-guide/interfaces/index.md)です。</span><span class="sxs-lookup"><span data-stu-id="083cf-105">For more information, see [Events](../../csharp/programming-guide/events/index.md) and [Interfaces](../../csharp/programming-guide/interfaces/index.md).</span></span>  
  
 <span data-ttu-id="083cf-106">次の例では CS0069 エラーが生成されます。</span><span class="sxs-lookup"><span data-stu-id="083cf-106">The following sample generates CS0069:</span></span>  
  
```csharp  
// CS0069.cs

public delegate void EventHandler();  
  
public interface a  
{  
   event EventHandler Click { remove {} }   // CS0069  
   event EventHandler Click2;   // OK  
}  
```
