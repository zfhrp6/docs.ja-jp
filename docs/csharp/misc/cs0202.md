# <a name="compiler-error-cs0202"></a><span data-ttu-id="8fab5-102">コンパイラ エラー CS0202</span><span class="sxs-lookup"><span data-stu-id="8fab5-102">Compiler Error CS0202</span></span>
<span data-ttu-id="8fab5-103">foreach では、'type.GetEnumerator()' の戻り値の型 'type' に適切なパブリック MoveNext メソッドおよびパブリック Current プロパティが含まれている必要があります</span><span class="sxs-lookup"><span data-stu-id="8fab5-103">foreach requires that the return type 'type' of 'type.GetEnumerator()' must have a suitable public MoveNext method and public Current property</span></span>  
  
 <span data-ttu-id="8fab5-104">foreach ステートメントを使用できるようにするために使用される <xref:System.Collections.IEnumerable.GetEnumerator%2A> 関数は、ポインターまたは配列を返すことはできません。列挙子として機能できるクラスのインスタンスを返す必要があります。</span><span class="sxs-lookup"><span data-stu-id="8fab5-104">A <xref:System.Collections.IEnumerable.GetEnumerator%2A> function, used to enable the use of foreach statements, cannot return a pointer or array; it must return an instance of a class that is able to act as an enumerator.</span></span> <span data-ttu-id="8fab5-105">列挙子として機能するための適切な要件には、パブリック Current プロパティおよびパブリック MoveNext メソッドが含まれます。</span><span class="sxs-lookup"><span data-stu-id="8fab5-105">The proper requirements to serve as an enumerator include a public Current property and a public MoveNext method.</span></span>  
  
> [!NOTE]
>  <span data-ttu-id="8fab5-106">C# 2.0 では、コンパイラによって Current および MoveNext が自動的に生成されます。</span><span class="sxs-lookup"><span data-stu-id="8fab5-106">In C# 2.0, the compiler will automatically generate Current and MoveNext for you.</span></span> <span data-ttu-id="8fab5-107">詳細についてでのコード例を参照してください。[ジェネリック インターフェイス](../../csharp/programming-guide/generics/generic-interfaces.md)です。</span><span class="sxs-lookup"><span data-stu-id="8fab5-107">For more information, see the code example in [Generic Interfaces](../../csharp/programming-guide/generics/generic-interfaces.md).</span></span>  
  
 <span data-ttu-id="8fab5-108">次の例では CS0202 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="8fab5-108">The following sample generates CS0202:</span></span>  
  
```csharp  
// CS0202.cs  
  
public class C1  
{  
   public int Current  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public bool MoveNext ()  
   {  
      return false;  
   }  
  
   public static implicit operator C1 (int c1)  
   {  
      return 0;  
   }  
}  
  
public class C2  
{  
   public int Current  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public bool MoveNext ()  
   {  
      return false;  
   }  
  
   public C1[] GetEnumerator ()  
   // try the following line instead  
   // public C1 GetEnumerator ()  
   {  
      return null;  
   }  
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
      C2 c2 = new C2();  
  
      foreach (C1 x in c2)   // CS0202  
      {  
         System.Console.WriteLine(x.Current);  
      }  
   }  
}  
```
