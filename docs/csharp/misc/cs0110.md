# <a name="compiler-error-cs0110"></a><span data-ttu-id="bd34c-102">コンパイラ エラー CS0110</span><span class="sxs-lookup"><span data-stu-id="bd34c-102">Compiler Error CS0110</span></span>
<span data-ttu-id="bd34c-103">'const declaration' の定数値の評価により、循環定義が発生します。</span><span class="sxs-lookup"><span data-stu-id="bd34c-103">The evaluation of the constant value for 'const declaration' involves a circular definition</span></span>  
  
 <span data-ttu-id="bd34c-104">[const](../../csharp/language-reference/keywords/const.md) 変数 (`a`) の宣言は、(`b`) も参照する別の const 変数 (`a`) を参照することはできません。</span><span class="sxs-lookup"><span data-stu-id="bd34c-104">The declaration of a [const](../../csharp/language-reference/keywords/const.md) variable (`a`) cannot reference another const variable (`b`) that also references (`a`).</span></span>  
  
 <span data-ttu-id="bd34c-105">次の例では CS0110 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="bd34c-105">The following sample generates CS0110:</span></span>  
  
```csharp  
// CS0110.cs  
namespace MyNamespace  
{  
   public class A  
   {  
      public static void Main()  
      {  
      }  
   }  
  
   public class B : A  
   {  
      public const int i = c + 1;   // CS0110, c already references i  
      public const int c = i + 1;  
      // the following line would be OK  
      // public const int c = 10;  
   }  
}  
```
