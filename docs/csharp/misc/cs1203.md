# <a name="compiler-warning-level-1-cs1203"></a>コンパイラの警告 (レベル 1) CS1203
機能 'feature' は推奨されていません。 代わりに 'feature' を使用してください。  
  
 機能 'invalid feature' は推奨されていません。 代わりに 'valid feature' を使用してください。  
  
 使用しようとした機能は、現在使用されていません。 代わりに、正しい機能を使用するようにコードを更新します。
