## <a name="example"></a><span data-ttu-id="d319f-109">例</span><span class="sxs-lookup"><span data-stu-id="d319f-109">Example</span></span>  
 <span data-ttu-id="d319f-110">このコードでは、部分クラス定義がすべて 1 つのファイル内に記述されていますが、複数のファイルに分けることもできます。</span><span class="sxs-lookup"><span data-stu-id="d319f-110">In this code, the partial class definitions are all in a single file, but they could also be spread across multiple files.</span></span>

```csharp  
{  
    interface IFace1 { }  
    interface IFace2 { }  
    partial class PartialBadBounds<T> where T : IFace1 { } // CS0265

}  
```
