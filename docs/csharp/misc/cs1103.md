# コンパイラ エラー CS1103
拡張メソッドの最初のパラメーターを型 'type' にすることはできません。  
  
 拡張メソッドの最初のパラメーターをポインター型にすることはできません。  
  
## 使用例  
 次の例では CS1103 が生成されます。  
  
```  
// cs1103.cs public static class Extensions { public unsafe static char* Test(this char* charP) { return charP; } // CS1103 }   
```
