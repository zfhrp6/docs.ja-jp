# <a name="compiler-error-cs1513"></a>コンパイラ エラー CS1513
} が必要です。  
  
 コンパイラには右中かっこ (`}`) が必要ですが、見つかりませんでした。  
  
 次の例では CS1513 が生成されます。  
  
```  
// CS1513  
namespace y   // CS1513, no close curly brace  
{  
   class x  
   {  
      public static void Main()  
      {  
      }  
   }  
```
