# <a name="compiler-warning-level-4-cs0628"></a><span data-ttu-id="cac06-102">コンパイラの警告 (レベル 4) CS0628</span><span class="sxs-lookup"><span data-stu-id="cac06-102">Compiler Warning (level 4) CS0628</span></span>
<span data-ttu-id="cac06-103">'member': 新規のプロテクト メンバーがシール クラスで宣言されました</span><span class="sxs-lookup"><span data-stu-id="cac06-103">'member' : new protected member declared in sealed class</span></span>  
  
 <span data-ttu-id="cac06-104">他のクラスは [クラスから継承できず、](../../csharp/language-reference/keywords/sealed.md) メンバーを使用することができないため、 [シール](../../csharp/language-reference/keywords/protected.md) クラスは `sealed` プロテクト `protected` メンバーを使用できません。</span><span class="sxs-lookup"><span data-stu-id="cac06-104">A [sealed](../../csharp/language-reference/keywords/sealed.md) class cannot introduce a [protected](../../csharp/language-reference/keywords/protected.md) member because no other class will be able to inherit from the `sealed` class and use the `protected` member.</span></span>  
  
 <span data-ttu-id="cac06-105">次の例では CS0628 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="cac06-105">The following sample generates CS0628:</span></span>

```csharp  
{  
   protected int i;   // CS0628  
}

{  
   public static void Main()  
   {  
   }  
}  
```
