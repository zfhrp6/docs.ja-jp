# <a name="compiler-error-cs1625"></a><span data-ttu-id="6aa08-102">コンパイラ エラー CS1625</span><span class="sxs-lookup"><span data-stu-id="6aa08-102">Compiler Error CS1625</span></span>
<span data-ttu-id="6aa08-103">finally 句の本体で生成することはできません。</span><span class="sxs-lookup"><span data-stu-id="6aa08-103">Cannot yield in the body of a finally clause</span></span>  
  
 <span data-ttu-id="6aa08-104">yield ステートメントは、finally 句の本体で使用することはできません。</span><span class="sxs-lookup"><span data-stu-id="6aa08-104">A yield statement is not allowed in the body of a finally clause.</span></span> <span data-ttu-id="6aa08-105">このエラーを回避するには、yield ステートメントを finally 句の外に移動します。</span><span class="sxs-lookup"><span data-stu-id="6aa08-105">To avoid this error, move the yield statement out of the finally clause.</span></span>  
  
 <span data-ttu-id="6aa08-106">次の例では CS1625 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="6aa08-106">The following sample generates CS1625:</span></span>  
  
```  
// CS1625.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
      }  
      finally  
      {  
        yield return this;  // CS1625  
      }  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
