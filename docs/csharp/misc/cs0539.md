# <a name="compiler-error-cs0539"></a><span data-ttu-id="70afd-102">コンパイラ エラー CS0539</span><span class="sxs-lookup"><span data-stu-id="70afd-102">Compiler Error CS0539</span></span>
<span data-ttu-id="70afd-103">明示的インターフェイス宣言の中の 'member' はインターフェイスのメンバーではありません</span><span class="sxs-lookup"><span data-stu-id="70afd-103">'member' in explicit interface declaration is not a member of interface</span></span>  
  
 <span data-ttu-id="70afd-104">存在しない [インターフェイス](../../csharp/language-reference/keywords/interface.md) メンバーを明示的に宣言しようとしました。</span><span class="sxs-lookup"><span data-stu-id="70afd-104">An attempt was made to explicitly declare an [interface](../../csharp/language-reference/keywords/interface.md) member that does not exist.</span></span> <span data-ttu-id="70afd-105">宣言を削除するか、有効なインターフェイス メンバーを参照するように宣言を変更する必要があります。</span><span class="sxs-lookup"><span data-stu-id="70afd-105">You should either delete the declaration or change it so that it refers to a valid interface member.</span></span>  
  
 <span data-ttu-id="70afd-106">次の例では CS0539 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="70afd-106">The following sample generates CS0539:</span></span>  
  
```csharp  
// CS0539.cs  
namespace x  
{  
   interface I  
   {  
      void m();  
   }  
  
   public class clx : I  
   {  
      void I.x()   // CS0539  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
