## <a name="example"></a><span data-ttu-id="15042-110">例</span><span class="sxs-lookup"><span data-stu-id="15042-110">Example</span></span>  
 <span data-ttu-id="15042-111">次のコードでは、 `Program.testMethod` が静的として宣言されるため、CS0736 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="15042-111">The following code generates CS0736 because `Program.testMethod` is declared as static:</span></span>  
  
```

namespace CS0736  
{     
  
    interface ITest  
    {  
        int testMethod(int x);  
    }  
  
    class Program : ITest // CS0736  
    {  
        public static int testMethod(int x) { return 0; }  
        // Try the following line instead.  
        // public int testMethod(int x) { return 0; }  
        public static void Main() { }  
    }      
}  
```
