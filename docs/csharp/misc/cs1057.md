# コンパイラ エラー CS1057
'member': 静的クラスにプロテクト メンバーを含めることはできません  
  
 このエラーは、静的クラス内でプロテクト メンバーを宣言すると生成されます。  
  
## 使用例  
 次の例では CS1057 が生成されます。  
  
```  
// CS1057.cs using System; static class Class1 { protected static int x;   // CS1057 public static void Main() { } }  
```
