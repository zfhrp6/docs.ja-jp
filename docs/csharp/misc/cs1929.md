# <a name="compiler-error-cs1929"></a><span data-ttu-id="b5992-102">コンパイラ エラー CS1929</span><span class="sxs-lookup"><span data-stu-id="b5992-102">Compiler Error CS1929</span></span>
<span data-ttu-id="b5992-103">インスタンス引数: 'typeA' から 'typeB' に変換できません。</span><span class="sxs-lookup"><span data-stu-id="b5992-103">Instance argument: cannot convert from 'typeA' to 'typeB'.</span></span>  
  
 <span data-ttu-id="b5992-104">このエラーは、拡張しないクラスから拡張メソッドを呼び出そうとすると生成されます。</span><span class="sxs-lookup"><span data-stu-id="b5992-104">This error is generated when you try to invoke an extension method from a class that it does not extend.</span></span> <span data-ttu-id="b5992-105">次に示す例では、拡張メソッドは派生クラス `A`には定義されますが、基底クラス `B`には定義されません。</span><span class="sxs-lookup"><span data-stu-id="b5992-105">In the example shown here, the extension method is defined for the derived class `A`, but not for the base class `B`.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="b5992-106">このエラーを解決するには</span><span class="sxs-lookup"><span data-stu-id="b5992-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="b5992-107">呼び出す必要のある型用の新しい拡張メソッドを作成するか、既存のメソッドを拡張する型のオブジェクトに呼び出しを移動します。</span><span class="sxs-lookup"><span data-stu-id="b5992-107">Create a new extension method for the type where you have to invoke it, or else move the call into an object of the type that the existing method extends.</span></span>  
  
## <a name="example"></a><span data-ttu-id="b5992-108">例</span><span class="sxs-lookup"><span data-stu-id="b5992-108">Example</span></span>  
 <span data-ttu-id="b5992-109">次のコードでは CS1928 および CS1929 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="b5992-109">The following code generates CS1928 and CS1929:</span></span>  
  
```csharp  
// cs1929.cs  
using System.Linq;  
    using System.Collections;  
  
    static class Ext  
    {  
        public static void ExtMethod(this A a)  
        {  
        }  
    }  
  
    class A : B  
    {  
    }  
  
    class B  
    {  
        static void Main()  
        {  
            B b = new B();  
            b.ExtMethod(); // CS1929  
        }  
    }  
```
