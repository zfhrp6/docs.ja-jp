 <span data-ttu-id="77d46-104">**event** 定義に含まれる **add** キーワードまたは [remove](../../csharp/language-reference/keywords/event.md) キーワードには本体が必要です。</span><span class="sxs-lookup"><span data-stu-id="77d46-104">An **add** or **remove** keyword in an [event](../../csharp/language-reference/keywords/event.md) definition must have a body.</span></span> <span data-ttu-id="77d46-105">詳細については、「[イベント](../../csharp/programming-guide/events/index.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="77d46-105">For more information, see [Events](../../csharp/programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="77d46-106">次の例では CS0073 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="77d46-106">The following sample generates CS0073:</span></span>  
  
```  
// CS0073.cs  
delegate void del();  
  
class Test  
{  
   public event del MyEvent  
   {  
      add;   // CS0073  
      // try the following lines instead  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      remove  
      {  
         MyEvent -= value;  
      }  
  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
