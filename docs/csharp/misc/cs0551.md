 <span data-ttu-id="63248-104">インターフェイスのプロパティを明示的に実装するクラスは、インターフェイスが定義するすべてのアクセサーを実装する必要があります。</span><span class="sxs-lookup"><span data-stu-id="63248-104">A class that explicitly implements an interface's property must implement all the accessors that the interface defines.</span></span>  
  
 <span data-ttu-id="63248-105">詳細については、「[プロパティの使用](../../csharp/programming-guide/classes-and-structs/using-properties.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="63248-105">For more information, see [Using Properties](../../csharp/programming-guide/classes-and-structs/using-properties.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="63248-106">例</span><span class="sxs-lookup"><span data-stu-id="63248-106">Example</span></span>  
 <span data-ttu-id="63248-107">次の例では CS0551 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="63248-107">The following sample generates CS0551.</span></span>  
  
```  
// CS0551.cs  
// compile with: /target:library  
interface ii  
{  
   int i  
   {  
      get;  
      set;  
   }  
}  
  
public class a : ii  
{  
   int ii.i { set {} }   // CS0551  
  
   // OK  
   int ii.i      
   {  
      set {}  
      get { return 0; }  
   }  
}  
```
