# <a name="compiler-error-cs0075"></a><span data-ttu-id="31474-102">コンパイラ エラー CS0075</span><span class="sxs-lookup"><span data-stu-id="31474-102">Compiler Error CS0075</span></span>
<span data-ttu-id="31474-103">負の値をキャストするには、値をかっこで囲んでください。</span><span class="sxs-lookup"><span data-stu-id="31474-103">To cast a negative value, you must enclose the value in parentheses</span></span>  
  
 <span data-ttu-id="31474-104">定義済みの型を指定するキーワードを使用してキャストしている場合、かっこを付ける必要はありません。</span><span class="sxs-lookup"><span data-stu-id="31474-104">If you are casting using a keyword that identifies a predefined type, then you do not need parentheses.</span></span> <span data-ttu-id="31474-105">それ以外の場合は、かっこを付ける必要があります。(x) –y がキャスト式と見なされないためです。</span><span class="sxs-lookup"><span data-stu-id="31474-105">Otherwise, you must put the parentheses because (x) –y will not be considered a cast expression.</span></span> <span data-ttu-id="31474-106">以下は「C# 言語の仕様」のセクション 7.6.6 からの抜粋です。</span><span class="sxs-lookup"><span data-stu-id="31474-106">From the C# Specification, Section 7.6.6:</span></span>  
  
 <span data-ttu-id="31474-107">*あいまいさを排除規則に従う場合から x および y 識別子、(x) y、(あり、および (とは、(x)、キャスト式-y が x 型を指定する場合でもは、します。ただし、x が定義済みの型 (int など) を指定するキーワードの場合は、し、4 つの形式はキャスト式 (このようなキーワード可能性のあるできなかったため、式を単独で)。*</span><span class="sxs-lookup"><span data-stu-id="31474-107">*From the disambiguation rule it follows that, if x and y are identifiers, (x)y, (x)(y), and (x)(-y) are cast-expressions, but (x)-y is not, even if x identifies a type. However, if x is a keyword that identifies a predefined type (such as int), then all four forms are cast-expressions (because such a keyword could not possibly be an expression by itself).*</span></span>  
  
 <span data-ttu-id="31474-108">次のコードでは CS0075 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="31474-108">The following code generates CS0075:</span></span>  
  
```csharp  
// CS0075  
namespace MyNamespace  
{  
    enum MyEnum { }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            // To fix the error, place the negative  
            // values below in parentheses  
            int i = (System.Int32) - 4; //CS0075  
            MyEnum e = (MyEnum) - 1;    //CS0075  
            System.Console.WriteLine(i); //to avoid warning  
            System.Console.WriteLine(e); //to avoid warning  
        }  
    }  
}  
```
