# <a name="compiler-warning-level-1-cs0658"></a>コンパイラの警告 (レベル 1) CS0658
'attribute modifier' は認識できる属性の場所ではありません。 このブロック内の属性はすべて無視されます。  
  
 無効な属性の修飾子が指定されました。 詳細については、「 [属性の対象](http://msdn.microsoft.com/en-us/59a261f0-1cfb-4aa5-b610-6b735389882c) 」を参照してください。  
  
 次の例では CS0658 が生成されます。  
  
```  
// CS0658.cs  
using System;  
public class TestAttribute : Attribute{}  
[badAttributeLocation: Test]   // CS0658, badAttributeLocation is invalid  
class ClassTest  
{  
   public static void Main()  
   {  
   }  
}  
```
