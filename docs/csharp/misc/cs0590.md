# <a name="compiler-error-cs0590"></a><span data-ttu-id="44f06-102">コンパイラ エラー CS0590</span><span class="sxs-lookup"><span data-stu-id="44f06-102">Compiler Error CS0590</span></span>

 <span data-ttu-id="44f06-104">ユーザー定義の演算子の目的は、オブジェクトを返すことです。</span><span class="sxs-lookup"><span data-stu-id="44f06-104">The purpose of a user-defined operator is to return an object.</span></span>  
  
 <span data-ttu-id="44f06-105">次の例では CS0590 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="44f06-105">The following sample generates CS0590:</span></span>  
  
```csharp  
// CS0590.cs  
namespace x  
{  
   public class a  
   {  
      public static void operator+(a A1, a A2)   // CS0590  
      {  
      }  
  
      // try the following user-defined operator  
      /*  
      public static a operator+(a A1, a A2)  
      {  
         return A2;  
      }  
      */  
  
      public static int Main()  
      {  
         return 1;  
      }  
   }  
}  
```
