# <a name="compiler-error-cs0432"></a><span data-ttu-id="f7930-102">コンパイラ エラー CS0432</span><span class="sxs-lookup"><span data-stu-id="f7930-102">Compiler Error CS0432</span></span>
<span data-ttu-id="f7930-103">エイリアス 'identifier' が見つかりません</span><span class="sxs-lookup"><span data-stu-id="f7930-103">Alias 'identifier' not found</span></span>  
  
 <span data-ttu-id="f7930-104">このエラーは、エイリアスではない識別子の右側に "::" を使用した場合に発生します。</span><span class="sxs-lookup"><span data-stu-id="f7930-104">This error occurs when you use "::" to the right of an identifier that is not an alias.</span></span> <span data-ttu-id="f7930-105">エラーを解決するには、代わりに "." を使用します。</span><span class="sxs-lookup"><span data-stu-id="f7930-105">To resolve the error, use "." instead.</span></span>  
  
 <span data-ttu-id="f7930-106">次の例では、CS0432 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="f7930-106">The following example generates CS0432:</span></span>  
  
```  
// CS0432.cs  
namespace A {  
    public class B {  
        public static void Meth() { }  
    }  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        A::B.Meth();   // CS0432  
       // To resolve, use the following line instead:  
       // A.B.Meth();  
    }  
}  
```
