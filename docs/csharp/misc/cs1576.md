# <a name="compiler-error-cs1576"></a><span data-ttu-id="363ce-102">コンパイラ エラー CS1576</span><span class="sxs-lookup"><span data-stu-id="363ce-102">Compiler Error CS1576</span></span>

 <span data-ttu-id="363ce-104">[#line](../../csharp/language-reference/preprocessor-directives/preprocessor-line.md) ディレクティブに渡された値にエラーが検出されました。</span><span class="sxs-lookup"><span data-stu-id="363ce-104">The compiler detected an error with the value passed to the [#line](../../csharp/language-reference/preprocessor-directives/preprocessor-line.md) directive.</span></span>

```csharp  
// CS1576.cs  
public class MyClass  
{  
   static void Main()  
   {  
      #line "abc.sc"         // CS1576  
      // try the following line instead  
      //#line  101 "abc.sc"  
      intt i;  // error will be reported on line 101  
   }  
}  
```
