# <a name="compiler-error-cs1511"></a><span data-ttu-id="239b9-102">コンパイラ エラー CS1511</span><span class="sxs-lookup"><span data-stu-id="239b9-102">Compiler Error CS1511</span></span>

 <span data-ttu-id="239b9-104">[base](../../csharp/language-reference/keywords/base.md) キーワードが [static](../../csharp/language-reference/keywords/static.md) メソッドで使用されました。</span><span class="sxs-lookup"><span data-stu-id="239b9-104">The [base](../../csharp/language-reference/keywords/base.md) keyword was used in a [static](../../csharp/language-reference/keywords/static.md) method.</span></span> <span data-ttu-id="239b9-105">`base` は、インスタンス コンストラクター、インスタンス メソッド、またはインスタンス アクセサーでのみ呼び出すことができます。</span><span class="sxs-lookup"><span data-stu-id="239b9-105">`base` can only be called in an instance constructor, instance method, or instance accessor.</span></span>  
  
## <a name="example"></a><span data-ttu-id="239b9-106">例</span><span class="sxs-lookup"><span data-stu-id="239b9-106">Example</span></span>  
 <span data-ttu-id="239b9-107">次の例では CS1511 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="239b9-107">The following sample generates CS1511.</span></span>  
  
```  
// CS1511.cs  
// compile with: /target:library  
public class A  
{  
   public int j = 0;  
}  
  
class C : A  
{  
   public void Method()  
   {  
      base.j = 3;   // base allowed here  
   }  
  
   public static int StaticMethod()  
   {  
      base.j = 3;   // CS1511  
      return 1;  
   }  
}  
```
