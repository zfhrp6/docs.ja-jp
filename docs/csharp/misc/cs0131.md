## <a name="example"></a><span data-ttu-id="b6633-108">例</span><span class="sxs-lookup"><span data-stu-id="b6633-108">Example</span></span>  
 <span data-ttu-id="b6633-109">次の例では CS0131 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="b6633-109">The following sample generates CS0131.</span></span>  
  
```csharp  
// CS0131.cs  
public class MyClass  
{  
    public int i = 0;  
    public void MyMethod()  
    {  
        i++ = 1;   // CS0131  
        // try the following line instead  
        // i = 1;  
    }  
    public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="b6633-110">例</span><span class="sxs-lookup"><span data-stu-id="b6633-110">Example</span></span>

```csharp  
// CS0131b.cs  
public class C  
{  
    public static int Main()  
    {  
        int a = 1, b = 2, c = 3;

        // try this instead  
        // if (a + b == c)

        return 1;  
    }

```
