# <a name="compiler-warning-level-1-cs1201"></a>コンパイラの警告 (レベル 1) CS1201
機能 'invalid feature' は使用できません。 'valid feature' を使用してください。  
  
 使用しようとした機能は、現在使用されていません。 代わりに、正しい機能を使用するようにコードを更新します。
