## <a name="example"></a><span data-ttu-id="3eee0-105">例</span><span class="sxs-lookup"><span data-stu-id="3eee0-105">Example</span></span>

```  
// CS0448.cs  
class C5  
{  
   public static int operator ++(C5 c) { return null; }   // CS0448

   public static void Main() {}  
}  
```
