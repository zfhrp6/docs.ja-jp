<span data-ttu-id="5ff10-103">++ または -- 演算子の戻り値の型は、それを含む型であるか、またはそれを含む型から派生しなければなりません</span><span class="sxs-lookup"><span data-stu-id="5ff10-103">The return type for ++ or -- operator must be the containing type or derived from the containing type</span></span>

## <a name="example"></a><span data-ttu-id="5ff10-105">例</span><span class="sxs-lookup"><span data-stu-id="5ff10-105">Example</span></span>  
 <span data-ttu-id="5ff10-106">次の例では CS0448 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="5ff10-106">The following sample generates CS0448.</span></span>  
  
```csharp  
// CS0448.cs  
class C5  
{  
   public static int operator ++(C5 c) { return null; }   // CS0448  
   public static C5 operator --(C5 c) { return null; }   // OK  
   public static void Main() {}  
}  
```

   public static T? operator ++(T? t) { return new T(); }
