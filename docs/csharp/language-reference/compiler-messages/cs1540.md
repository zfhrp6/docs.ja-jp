# <a name="compiler-error-cs1540"></a><span data-ttu-id="beb6c-102">コンパイラ エラー CS1540</span><span class="sxs-lookup"><span data-stu-id="beb6c-102">Compiler Error CS1540</span></span>
<span data-ttu-id="beb6c-103">'<型 1>' 型の修飾子をとおしてプロテクト メンバー '<メンバー>' にアクセスすることはできません。修飾子は '<型 2>' 型、またはそれから派生したものでなければなりません。</span><span class="sxs-lookup"><span data-stu-id="beb6c-103">Cannot access protected member 'member' via a qualifier of type 'type1'; the qualifier must be of type 'type2' (or derived from it)</span></span>  
  
 <span data-ttu-id="beb6c-104">派生[クラス](../../../csharp/language-reference/keywords/class.md)は、基本クラスのインスタンスを通じて基本クラスのプロテクト メンバーにアクセスすることはできません。</span><span class="sxs-lookup"><span data-stu-id="beb6c-104">A derived [class](../../../csharp/language-reference/keywords/class.md) cannot access protected members of its base class through an instance of the base class.</span></span> <span data-ttu-id="beb6c-105">派生クラス内で宣言された基本クラスのインスタンスが、実行時に、派生元が同じであること以外には派生クラスとの関連性のない別の型のインスタンスになることがあります。</span><span class="sxs-lookup"><span data-stu-id="beb6c-105">An instance of the base class declared in the derived class might, at run time, be an instance of another type that is derived from the same base but is not otherwise related to the derived class.</span></span> <span data-ttu-id="beb6c-106">プロテクト メンバーにアクセスできるのは派生した型のみであるため、実行時に有効でない可能性があるプロテクト メンバーにアクセスする試みは、コンパイラによって無効とマークされます。</span><span class="sxs-lookup"><span data-stu-id="beb6c-106">Because protected members can be accessed only by derived types, any attempts to access protected members that might not be valid at run time are marked by the compiler as not valid.</span></span>  
  
 <span data-ttu-id="beb6c-107">次の例の `Employee` クラスでは、`emp2` と `emp3` の両方がコンパイル時に `Person` 型になりますが、`emp2` は実行時に `Manager` 型になります。</span><span class="sxs-lookup"><span data-stu-id="beb6c-107">In the `Employee` class in the following example, `emp2` and `emp3` both have type `Person` at compile time, but `emp2` has type `Manager` at run time.</span></span> <span data-ttu-id="beb6c-108">`Employee` は `Manager` から派生しないため、基本クラスである `Person` のプロテクト メンバーに、`Manager` クラスのインスタンスを通じてアクセスすることはできません。</span><span class="sxs-lookup"><span data-stu-id="beb6c-108">Because `Employee` is not derived from `Manager`, it cannot access the protected members of the base class, `Person`, through an instance of the `Manager` class.</span></span> <span data-ttu-id="beb6c-109">コンパイラは、2 つの `Person` オブジェクトの実行時の型が何になるかを判断できません。</span><span class="sxs-lookup"><span data-stu-id="beb6c-109">The compiler cannot determine what the run-time type of the two `Person` objects will be.</span></span> <span data-ttu-id="beb6c-110">したがって、`emp2` からの呼び出しと `emp3` からの呼び出しによってコンパイラ エラー CS1540 が発生します。</span><span class="sxs-lookup"><span data-stu-id="beb6c-110">Therefore, both the call from `emp2` and the call from `emp3` cause compiler error CS1540.</span></span>  
  
```csharp  
using System;  
using System.Text;  
  
namespace CS1540  
{  
    class Program1  
    {  
        static void Main()  
        {  
            Employee.PreparePayroll();  
        }  
    }  
  
    class Person  
    {  
        protected virtual void CalculatePay()   
        {  
            Console.WriteLine("CalculatePay in Person class.");  
        }  
    }  
  
    class Manager : Person  
    {  
        protected override void CalculatePay()   
        {  
            Console.WriteLine("CalculatePay in Manager class.");   
  
        }  
    }  
  
    class Employee : Person  
    {  
        public static void PreparePayroll()  
        {  
            Employee emp1 = new Employee();  
            Person emp2 = new Manager();  
            Person emp3 = new Employee();  
            // The following line calls the method in the Employee base class,  
            // Person.  
            emp1.CalculatePay();   
  
            // The following lines cause compiler error CS1540. The compiler   
            // cannot determine at compile time what the run-time types of   
            // emp2 and emp3 will be.  
            //emp2.CalculatePay();   
            //emp3.CalculatePay();  
  
        }  
    }  
}  
```
