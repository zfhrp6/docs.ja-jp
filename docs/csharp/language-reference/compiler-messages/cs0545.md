# <a name="compiler-error-cs0545"></a><span data-ttu-id="4ec80-102">コンパイラ エラー CS0545</span><span class="sxs-lookup"><span data-stu-id="4ec80-102">Compiler Error CS0545</span></span>

 <span data-ttu-id="4ec80-104">基底クラスにこのようなオーバーライドする定義がない場合に、プロパティ アクセサーのオーバーライドを定義する処理が試行されました。</span><span class="sxs-lookup"><span data-stu-id="4ec80-104">A try was made to define an override for a property accessor when the base class has no such definition to override.</span></span> <span data-ttu-id="4ec80-105">このエラーは、次の方法で解決できます。</span><span class="sxs-lookup"><span data-stu-id="4ec80-105">You can resolve this error by:</span></span>  
  
-   <span data-ttu-id="4ec80-106">基底クラスに `set` アクセサーを追加します。</span><span class="sxs-lookup"><span data-stu-id="4ec80-106">Adding a `set` accessor in the base class.</span></span>  
  
-   <span data-ttu-id="4ec80-107">派生クラスから `set` アクセサーを削除します。</span><span class="sxs-lookup"><span data-stu-id="4ec80-107">Removing the `set` accessor from the derived class.</span></span>  
  
-   <span data-ttu-id="4ec80-108">[new](../../../csharp/language-reference/keywords/new.md) キーワードを派生クラスのプロパティに追加することで、基底クラス プロパティを非表示にします。</span><span class="sxs-lookup"><span data-stu-id="4ec80-108">Hiding the base class property by adding the [new](../../../csharp/language-reference/keywords/new.md) keyword to a property in a derived class.</span></span>  
  
-   <span data-ttu-id="4ec80-109">基底クラスのプロパティを [virtual](../../../csharp/language-reference/keywords/virtual.md) にします。</span><span class="sxs-lookup"><span data-stu-id="4ec80-109">Making the base class property [virtual](../../../csharp/language-reference/keywords/virtual.md).</span></span>  
  
 <span data-ttu-id="4ec80-110">詳細については、「[プロパティの使用](../../../csharp/programming-guide/classes-and-structs/using-properties.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="4ec80-110">For more information, see [Using Properties](../../../csharp/programming-guide/classes-and-structs/using-properties.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="4ec80-111">例</span><span class="sxs-lookup"><span data-stu-id="4ec80-111">Example</span></span>  
 <span data-ttu-id="4ec80-112">次の例では CS0545 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="4ec80-112">The following sample generates CS0545.</span></span>  
  
```csharp  
// CS0545.cs  
// compile with: /target:library  
// CS0545  
public class a  
{  
   public virtual int i  
   {  
      set {}  
  
      // Uncomment the following line to resolve.  
      // get { return 0; }  
   }  
}  
  
public class b : a  
{  
   public override int i  
   {  
      get { return 0; }  
      set {}   // OK  
   }  
}  
```
