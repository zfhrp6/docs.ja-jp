# <a name="compiler-warning-level-1-cs0420"></a><span data-ttu-id="8b2f8-102">コンパイラの警告 (レベル 1) CS0420</span><span class="sxs-lookup"><span data-stu-id="8b2f8-102">Compiler Warning (level 1) CS0420</span></span>
<span data-ttu-id="8b2f8-103">'identifier': volatile フィールドへの参照は、volatile として扱われません。</span><span class="sxs-lookup"><span data-stu-id="8b2f8-103">'identifier': a reference to a volatile field will not be treated as volatile</span></span>  
  
 <span data-ttu-id="8b2f8-104">通常、volatile フィールドは `ref` または **out** パラメーターを使用して渡すことは推奨されません。これは、関数のスコープ内では volatile として扱われないためです。</span><span class="sxs-lookup"><span data-stu-id="8b2f8-104">A volatile field should not normally be passed using a `ref` or **out** parameter, since it will not be treated as volatile within the scope of the function.</span></span> <span data-ttu-id="8b2f8-105">この処理には、インターロック API の呼び出し時など、例外があります。</span><span class="sxs-lookup"><span data-stu-id="8b2f8-105">There are exceptions to this, such as when calling an interlocked API.</span></span> <span data-ttu-id="8b2f8-106">他の警告と同様に、まれではありますが参照パラメーターとして volatile フィールドを意図的に使用している場合、[#pragma warning](../../../csharp/language-reference/preprocessor-directives/preprocessor-pragma-warning.md) を使用してこの警告を無効にすることができます。</span><span class="sxs-lookup"><span data-stu-id="8b2f8-106">As with any warning, you may use the [#pragma warning](../../../csharp/language-reference/preprocessor-directives/preprocessor-pragma-warning.md) to disable this warning in those rare cases where you are intentionally using a volatile field as a reference parameter.</span></span>  
  
 <span data-ttu-id="8b2f8-107">次の例では CS0420 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="8b2f8-107">The following sample generates CS0420:</span></span>  
  
```csharp  
// CS0420.cs  
// compile with: /W:1  
using System;  
  
class TestClass  
{  
   private volatile int i;  
  
   public void TestVolatile(ref int ii)  
   {  
   }  
  
   public static void Main()  
   {  
      TestClass x = new TestClass();  
      x.TestVolatile(ref x.i);   // CS0420   
   }  
}  
```
