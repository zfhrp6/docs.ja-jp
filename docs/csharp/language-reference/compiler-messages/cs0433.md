# <a name="compiler-error-cs0433"></a><span data-ttu-id="03dc6-102">コンパイラ エラー CS0433</span><span class="sxs-lookup"><span data-stu-id="03dc6-102">Compiler Error CS0433</span></span>
<span data-ttu-id="03dc6-103">型 'TypeName1' が 'TypeName2' と 'TypeName3' の両方に存在します</span><span class="sxs-lookup"><span data-stu-id="03dc6-103">The type TypeName1 exists in both TypeName2 and TypeName3</span></span>  
  
 <span data-ttu-id="03dc6-104">アプリケーション内で参照される 2 つの異なるアセンブリに同じ名前空間と型が含まれているため、あいまいさが生じています。</span><span class="sxs-lookup"><span data-stu-id="03dc6-104">Two different assemblies referenced in your application contain the same namespace and type, which produces ambiguity.</span></span>  
  
 <span data-ttu-id="03dc6-105">このエラーを解決するには、「[/reference (C# コンパイラ オプション)](../../../csharp/language-reference/compiler-options/reference-compiler-option.md)」のコンパイラ オプションのエイリアス機能を使用するか、アセンブリのいずれかを参照しないようにします。</span><span class="sxs-lookup"><span data-stu-id="03dc6-105">To resolve this error, use the alias feature of the [/reference (C# Compiler Options)](../../../csharp/language-reference/compiler-options/reference-compiler-option.md) compiler option or do not reference one of your assemblies.</span></span>  
  
## <a name="example"></a><span data-ttu-id="03dc6-106">例</span><span class="sxs-lookup"><span data-stu-id="03dc6-106">Example</span></span>

```csharp

namespace TypeBindConflicts   
{  
   public class AggPubImpAggPubImp {}  
}  
```

```csharp  
namespace TypeBindConflicts   
{  
   public class AggPubImpAggPubImp {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="03dc6-110">例</span><span class="sxs-lookup"><span data-stu-id="03dc6-110">Example</span></span>  
 <span data-ttu-id="03dc6-111">次の例では CS0433 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="03dc6-111">The following example generates CS0433.</span></span>  
  
```csharp  
// CS0433_3.cs  
// compile with: /reference:cs0433_1.dll /reference:cs0433_2.dll  
using TypeBindConflicts;  
public class Test   
{  
   public static void Main()   
   {  
      AggPubImpAggPubImp n6 = new AggPubImpAggPubImp();   // CS0433  
   }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="03dc6-112">例</span><span class="sxs-lookup"><span data-stu-id="03dc6-112">Example</span></span>  
 <span data-ttu-id="03dc6-113">**/reference** コンパイラ オプションのエイリアス機能を使用して、この CS0433 エラーを解決する例を次に示します。</span><span class="sxs-lookup"><span data-stu-id="03dc6-113">The following example shows how you can use the alias feature of the **/reference** compiler option to resolve this CS0433 error.</span></span>  
  
```csharp  
// CS0433_4.cs  
// compile with: /reference:cs0433_1.dll /reference:TypeBindConflicts=cs0433_2.dll  
using TypeBindConflicts;

{  
   public static void Main()   
   {  
      AggPubImpAggPubImp n6 = new AggPubImpAggPubImp();  
   }  
}  
```
