# <a name="compiler-warning-level-1-cs1616"></a><span data-ttu-id="2abba-102">コンパイラの警告 (レベル 1) CS1616</span><span class="sxs-lookup"><span data-stu-id="2abba-102">Compiler Warning (level 1) CS1616</span></span>
<span data-ttu-id="2abba-103">オプション 'option' は、ソース ファイルまたは追加されたモジュールで指定された属性 'attribute' をオーバーライドします</span><span class="sxs-lookup"><span data-stu-id="2abba-103">Option 'option' overrides attribute 'attribute' given in a source file or added module</span></span>  
  
 <span data-ttu-id="2abba-104">この警告は、ソースで見つかったアセンブリ属性 <xref:System.Reflection.AssemblyKeyFileAttribute> または <xref:System.Reflection.AssemblyKeyNameAttribute> が、コマンド ライン オプション [/keyfile](../../../csharp/language-reference/compiler-options/keyfile-compiler-option.md) または [/keycontainer](../../../csharp/language-reference/compiler-options/keycontainer-compiler-option.md) と競合している場合、またはプロジェクトのプロパティで指定されたキー ファイル名またはキー コンテナーと競合している場合に発生します。</span><span class="sxs-lookup"><span data-stu-id="2abba-104">This warning occurs if the assembly attributes <xref:System.Reflection.AssemblyKeyFileAttribute> or <xref:System.Reflection.AssemblyKeyNameAttribute> found in source conflict with the [/keyfile](../../../csharp/language-reference/compiler-options/keyfile-compiler-option.md) or [/keycontainer](../../../csharp/language-reference/compiler-options/keycontainer-compiler-option.md) command line option or key file name or key container specified in the Project Properties.</span></span>  
  
 <span data-ttu-id="2abba-105">次の例では、`cs1616.snk` というキー ファイルを想定しています。</span><span class="sxs-lookup"><span data-stu-id="2abba-105">For the example below, assume you have a key file named `cs1616.snk`.</span></span> <span data-ttu-id="2abba-106">このファイルは、コマンド ラインで次のように入力して生成します。</span><span class="sxs-lookup"><span data-stu-id="2abba-106">Generate this file with the command line:</span></span>  
  
```console  
sn -k CS1616.snk  
```  
  
 <span data-ttu-id="2abba-107">次の例では CS1616 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="2abba-107">The following sample generates CS1616:</span></span>  
  
```csharp  
// CS1616.cs  
// compile with: /keyfile:cs1616.snk  
using System.Reflection;  
  
// To fix the error, remove the next line  
[assembly: AssemblyKeyFile("cs1616b.snk")]  // CS1616  
  
class C  
{  
  public static void Main()  
  {  
  }  
}  
```
