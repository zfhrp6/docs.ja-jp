 <span data-ttu-id="10f9e-105">通常のプロパティが `abstract` または `extern` としてマークされていない場合、`partial` 型のメンバーでない場合は、本体を指定する必要があります。</span><span class="sxs-lookup"><span data-stu-id="10f9e-105">Unless a regular property is marked as `abstract` or `extern`, or is a member of a `partial` type, it must supply a body.</span></span> <span data-ttu-id="10f9e-106">自動実装されるプロパティはアクセサー本体を提供しませんが、両方のアクセサーを指定する必要があります。</span><span class="sxs-lookup"><span data-stu-id="10f9e-106">Auto-implemented properties do not provide accessor bodies, but they must specify both accessors.</span></span> <span data-ttu-id="10f9e-107">読み取り専用の自動実装プロパティを作成するには、set アクセサーを `private` にします。</span><span class="sxs-lookup"><span data-stu-id="10f9e-107">To create a read-only auto-implemented property, make the set accessor `private`.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="10f9e-108">このエラーを解決するには</span><span class="sxs-lookup"><span data-stu-id="10f9e-108">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="10f9e-110">例</span><span class="sxs-lookup"><span data-stu-id="10f9e-110">Example</span></span>
