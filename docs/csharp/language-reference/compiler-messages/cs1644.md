<span data-ttu-id="7bbbf-103">機能 'feature' は標準 ISO C# 言語仕様の一部ではありません。別のコンパイラでは受け入れられない可能性があります</span><span class="sxs-lookup"><span data-stu-id="7bbbf-103">Feature 'feature' is not part of the standardized ISO C# language specification, and may not be accepted by other compilers</span></span>  
  
 <span data-ttu-id="7bbbf-104">このエラーは、[/langversion](../../../csharp/language-reference/compiler-options/langversion-compiler-option.md) オプションに ISO-1 を指定し、コンパイルしているコードで ISO 1.0 規格に準拠していない機能が使われている場合に発生します。</span><span class="sxs-lookup"><span data-stu-id="7bbbf-104">This error occurs if you specified the [/langversion](../../../csharp/language-reference/compiler-options/langversion-compiler-option.md) option ISO-1 and the code you are compiling uses features that are not part of the ISO 1.0 standard.</span></span> <span data-ttu-id="7bbbf-105">このエラーを解決するには、ISO-1 互換オプションを指定するときは、ジェネリックや匿名メソッドなどの C# 2.0 コンパイラの機能を一切使わないようにする必要があります。</span><span class="sxs-lookup"><span data-stu-id="7bbbf-105">To resolve this error, do not use any of the C# 2.0 compiler features such as generics or anonymous methods with the ISO-1 compatibility option.</span></span>
