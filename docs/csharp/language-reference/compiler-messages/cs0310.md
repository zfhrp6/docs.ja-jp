# <a name="compiler-error-cs0310"></a><span data-ttu-id="28246-102">コンパイラ エラー CS0310</span><span class="sxs-lookup"><span data-stu-id="28246-102">Compiler Error CS0310</span></span>
<span data-ttu-id="28246-103">'typename' は、ジェネリック型またはメソッド 'generic' 内でパラメーター 'parameter' として使用するために、パブリック パラメーターなしのコンストラクターを持つ非抽象型でなければなりません</span><span class="sxs-lookup"><span data-stu-id="28246-103">The type 'typename' must be a non-abstract type with a public parameterless constructor in order to use it as parameter 'parameter' in the generic type or method 'generic'</span></span>  
  
 <span data-ttu-id="28246-104">ジェネリック型またはメソッドは where 句で新しい制約を定義するため、そのジェネリック型またはメソッドに型引数として使用するには、どの型でもパラメーターなしのパブリック コンストラクターがある必要があります。</span><span class="sxs-lookup"><span data-stu-id="28246-104">The generic type or method defines a new constraint in its where clause, so any type must have a public parameterless constructor in order to be used as a type argument for that generic type or method.</span></span> <span data-ttu-id="28246-105">このエラーを回避するには、型のコンストラクターが正しいことを確認するか、ジェネリック型またはメソッドの制約句を修正します。</span><span class="sxs-lookup"><span data-stu-id="28246-105">To avoid this error, make sure that the type has the correct constructor, or modify the constraint clause of the generic type or method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="28246-106">例</span><span class="sxs-lookup"><span data-stu-id="28246-106">Example</span></span>  
 <span data-ttu-id="28246-107">次の例では CS0310 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="28246-107">The following sample generates CS0310:</span></span>  
  
```csharp  
// CS0310.cs  
using System;  
  
class G<T> where T : new()  
{  
    T t;  
  
    public G()  
    {  
        t = new T();  
        Console.WriteLine(t);  
    }  
}  
  
class B  
{  
    private B() { }  
    // Try this instead:  
    // public B() { }  
}  
  
class CMain  
{  
    public static void Main()  
    {  
        G<B> g = new G<B>();   // CS0310  
        Console.WriteLine(g.ToString());  
    }  
}  
```
