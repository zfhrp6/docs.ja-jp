# <a name="compiler-warning-level-2-cs0618"></a><span data-ttu-id="179bf-102">コンパイラの警告 (レベル 2) CS0618</span><span class="sxs-lookup"><span data-stu-id="179bf-102">Compiler Warning (level 2) CS0618</span></span>
<span data-ttu-id="179bf-103">'member' は使用されなくなりました: 'text'</span><span class="sxs-lookup"><span data-stu-id="179bf-103">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="179bf-104">クラス メンバーは `Obsolete` 属性でマークされているため、クラス メンバーを参照すると警告が発生します。</span><span class="sxs-lookup"><span data-stu-id="179bf-104">A class member was marked with the `Obsolete` attribute, such that a warning will be issued when the class member is referenced.</span></span> <span data-ttu-id="179bf-105">詳細については、「[共通の属性](../../programming-guide/concepts/attributes/common-attributes.md)」をご覧ください。</span><span class="sxs-lookup"><span data-stu-id="179bf-105">For more information, see [Common Attributes](../../programming-guide/concepts/attributes/common-attributes.md).</span></span>  
  
 <span data-ttu-id="179bf-106">次の例では CS0618 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="179bf-106">The following sample generates CS0618:</span></span>  
  
```csharp  
// CS0618.cs  
// compile with: /W:2  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", false)]   // warn if referenced  
   public static void m2()  
   {  
   }  
  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m2();  // CS0618  
   }  
}  
```
