### <a name="example-of-using-both-styles"></a><span data-ttu-id="d66ed-113">両方のスタイルの使用例</span><span class="sxs-lookup"><span data-stu-id="d66ed-113">Example of Using Both Styles</span></span>  
 <span data-ttu-id="d66ed-114">XML Web サービスを作成する場合、両方の属性セットをメソッドで使用できます。</span><span class="sxs-lookup"><span data-stu-id="d66ed-114">When you're creating an XML Web service, you can use both sets of attributes on the methods.</span></span> <span data-ttu-id="d66ed-115">次のコード例では、`MyService` という名前のクラスに、`MyLiteralMethod` と `MyEncodedMethod` という 2 つの XML Web サービス メソッドが含まれています。</span><span class="sxs-lookup"><span data-stu-id="d66ed-115">In the following code example, the class named `MyService` contains two XML Web service methods, `MyLiteralMethod` and `MyEncodedMethod`.</span></span> <span data-ttu-id="d66ed-116">いずれのメソッドも、`Order` クラスのインスタンスを返すという同じ機能を実行します。</span><span class="sxs-lookup"><span data-stu-id="d66ed-116">Both methods perform the same function: returning an instance of the `Order` class.</span></span> <span data-ttu-id="d66ed-117">`Order` クラスでは、<xref:System.Xml.Serialization.XmlTypeAttribute> 属性と <xref:System.Xml.Serialization.SoapTypeAttribute> 属性の両方が `OrderID` フィールドに適用され、両方の属性の `ElementName` プロパティには異なる値が設定されます。</span><span class="sxs-lookup"><span data-stu-id="d66ed-117">In the `Order` class, the <xref:System.Xml.Serialization.XmlTypeAttribute> and the <xref:System.Xml.Serialization.SoapTypeAttribute> attributes are both applied to the `OrderID` field, and both attributes have their `ElementName` property set to different values.</span></span>  
  
 <span data-ttu-id="d66ed-118">この例を実行するには、拡張子 .asmx のファイルにコードを貼り付け、そのファイルをインターネット インフォメーション サービス (IIS) によって管理される仮想ディレクトリに配置して、</span><span class="sxs-lookup"><span data-stu-id="d66ed-118">To run the example, paste the code into a file with an .asmx extension, and place the file into a virtual directory managed by Internet Information Services (IIS).</span></span> <span data-ttu-id="d66ed-119">Internet Explorer などの HTML ブラウザーから、コンピューター名、仮想ディレクトリ名、およびファイル名を入力します。</span><span class="sxs-lookup"><span data-stu-id="d66ed-119">From an HTML browser, such as Internet Explorer, type the name of the computer, virtual directory, and file.</span></span>  
  
```vb  
<%@ WebService Language="VB" Class="MyService" %>  
Imports System  
Imports System.Web.Services  
Imports System.Web.Services.Protocols  
Imports System.Xml.Serialization  
Public Class Order  
    ' Both types of attributes can be applied. Depending on which type  
    ' the method used, either one will affect the call.  
    <SoapElement(ElementName:= "EncodedOrderID"), _  
    XmlElement(ElementName:= "LiteralOrderID")> _  
    public OrderID As String  
End Class  
  
Public Class MyService  
    <WebMethod, SoapDocumentMethod> _  
    public Function MyLiteralMethod() As Order   
        Dim myOrder As Order = New Order()  
        return myOrder  
    End Function  
    <WebMethod, SoapRpcMethod> _  
    public Function MyEncodedMethod() As Order   
        Dim myOrder As Order = New Order()  
        return myOrder  
    End Function  
End Class  
```  
  
```csharp  
<%@ WebService Language="C#" Class="MyService" %>  
using System;  
using System.Web.Services;  
using System.Web.Services.Protocols;  
using System.Xml.Serialization;  
public class Order{  
    // Both types of attributes can be applied. Depending on which type  
    // the method used, either one will affect the call.  
    [SoapElement(ElementName = "EncodedOrderID")]  
    [XmlElement(ElementName = "LiteralOrderID")]  
    public String OrderID;  
}  
public class MyService{  
    [WebMethod][SoapDocumentMethod]  
    public Order MyLiteralMethod(){  
        Order myOrder = new Order();  
        return myOrder;  
    }  
    [WebMethod][SoapRpcMethod]  
    public Order MyEncodedMethod(){  
        Order myOrder = new Order();  
        return myOrder;  
    }  
}  
```  
  
 <span data-ttu-id="d66ed-120">`MyLiteralMethod` を呼び出すコード例を次に示します。</span><span class="sxs-lookup"><span data-stu-id="d66ed-120">The following code example calls `MyLiteralMethod`.</span></span> <span data-ttu-id="d66ed-121">要素名は "LiteralOrderID" に変更されています。</span><span class="sxs-lookup"><span data-stu-id="d66ed-121">The element name is changed to "LiteralOrderID".</span></span>  
  
```xml  
<?xml version="1.0" encoding="utf-8"?>  
<soap:Envelope xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xmlns:xsd="http://www.w3.org/2001/XMLSchema" xmlns:soap="http://schemas.xmlsoap.org/soap/envelope/">  
    <soap:Body>  
        <MyLiteralMethodResponse xmlns="http://tempuri.org/">  
            <MyLiteralMethodResult>  
                <LiteralOrderID>string</LiteralOrderID>  
            </MyLiteralMethodResult>  
        </MyLiteralMethodResponse>  
    </soap:Body>  
</soap:Envelope>  
```  
  
 <span data-ttu-id="d66ed-122">`MyEncodedMethod` を呼び出すコード例を次に示します。</span><span class="sxs-lookup"><span data-stu-id="d66ed-122">The following code example calls `MyEncodedMethod`.</span></span> <span data-ttu-id="d66ed-123">要素名は "EncodedOrderID" です。</span><span class="sxs-lookup"><span data-stu-id="d66ed-123">The element name is "EncodedOrderID".</span></span>  
  
```xml  
<?xml version="1.0" encoding="utf-8"?>  
<soap:Envelope xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xmlns:xsd="http://www.w3.org/2001/XMLSchema" xmlns:soapenc="http://schemas.xmlsoap.org/soap/encoding/" xmlns:tns="http://tempuri.org/" xmlns:types="http://tempuri.org/encodedTypes" xmlns:soap="http://schemas.xmlsoap.org/soap/envelope/">  
    <soap:Body soap:encodingStyle="http://schemas.xmlsoap.org/soap/encoding/">  
        <tns:MyEncodedMethodResponse>  
            <MyEncodedMethodResult href="#id1" />  
        </tns:MyEncodedMethodResponse>  
        <types:Order id="id1" xsi:type="types:Order">  
            <EncodedOrderID xsi:type="xsd:string">string</EncodedOrderID>  
        </types:Order>
```  
